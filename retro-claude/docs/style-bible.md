# Monster Chronicle スタイルバイブル

## 1. カラーパレット

### マスターパレット (32色)

GBC互換パレットをベースに、以下のサブパレットに分割して運用する。

```
BG パレット (8パレット × 4色 = 32色)
├── PAL_BG_0: 拠点タウン用       (#0F380F, #306230, #8BAC0F, #9BBC0F)
├── PAL_BG_1: 森ダンジョン用     (#1B2632, #2B4F3B, #5A8C5A, #A8D8A8)
├── PAL_BG_2: 洞窟ダンジョン用   (#1B1B2F, #3B3B5C, #7B7BA8, #C8C8E8)
├── PAL_BG_3: 火山ダンジョン用   (#2B0F0F, #5C1A1A, #B83B3B, #E88B5C)
├── PAL_BG_4: 氷ダンジョン用     (#0F1B2B, #1A3B5C, #5C8BAC, #A8D8E8)
├── PAL_BG_5: UI用               (#0F0F0F, #3B3B3B, #8B8B8B, #F8F8F8)
├── PAL_BG_6: 予備1
└── PAL_BG_7: 予備2

SPR パレット (8パレット × 3色+透過)
├── PAL_SPR_0: 主人公用
├── PAL_SPR_1: NPC用
├── PAL_SPR_2: モンスター暖色系
├── PAL_SPR_3: モンスター寒色系
├── PAL_SPR_4: モンスター緑系
├── PAL_SPR_5: モンスター紫/闇系
├── PAL_SPR_6: エフェクト用
└── PAL_SPR_7: 予備
```

**注意:** 実際のHex値はPhase 0で確定する。上記はプレースホルダー。
Lospec (https://lospec.com/palette-list) から GBC互換パレットを選定すること。

### パレット運用ルール

- 全アセットはマスターパレットの色のみ使用
- AI生成後は必ず Aseprite でパレットリマップを実行
- パレット外の色が1pxでもあれば不合格

## 2. スプライト仕様

### 共通ルール

| ルール | 仕様 | 厳守度 |
|--------|------|--------|
| アウトライン | 1px黒アウトライン統一 | 必須 |
| シェーディング | 2トーンセルシェーディング | 必須 |
| ディザリング | 使わない | 必須 |
| アンチエイリアシング | 使わない | 必須 |
| ミクセル | 禁止（ピクセル密度統一） | 必須 |
| サブピクセル | 使わない | 必須 |

### フィールドキャラクター

- サイズ: 16×16px
- プロポーション: 2頭身チビキャラ
- 方向: 上/下/左/右 の4方向
- アニメーション: 歩行2フレーム/方向 = 合計8フレーム
- 左右反転: 左向きは右向きのフリップで対応可

```
スプライトシート配置:
[下1][下2][左1][左2][右1][右2][上1][上2]
 16px × 8 = 128px 幅のスプライトシート
```

### バトルスプライト

| ランク | サイズ | 待機フレーム | 攻撃フレーム |
|--------|--------|-------------|-------------|
| E (小型) | 24×24px | 2 | - |
| D-C (中型) | 32×32px | 2 | - |
| B-A (大型) | 48×48px | 2 | - |
| S (ボス級) | 56×56px | 2 | 3 |

- 正面向き固定（DQM準拠）
- 待機アニメーションは上下の微動 or 明滅
- 攻撃アニメーションはS級のみ

### タイルセット

- グリッド: 8×8px
- タイル間の継ぎ目が自然に繋がること
- オートタイル対応を想定した配置

### UIパーツ

- メッセージウィンドウ: 9-slice対応 (角8×8px)
- カーソル: 8×8px, 2フレームアニメーション
- HP/MPバー: 高さ4px, 幅可変
- アイテムアイコン: 16×16px

## 3. アニメーション仕様

| アニメーション | フレーム数 | フレーム間隔 |
|---------------|-----------|-------------|
| 歩行サイクル | 2F/方向 | 200ms |
| 待機揺れ | 2F | 500ms |
| バトル攻撃(S級) | 3F | 150ms |
| カーソル点滅 | 2F | 300ms |
| ダメージフラッシュ | - | 100ms × 3回 |
| テキスト表示 | 1文字ずつ | 50ms/文字 |

## 5. ファイル命名規則

```
キャラクター:  chr_{name}_{direction}_{frame}.png
               chr_hero_down_01.png

モンスター:    mon_{id}_{name}_{state}_{frame}.png
               mon_001_slimeel_idle_01.png

タイルセット:  tile_{theme}_{type}.png
               tile_forest_floor_01.png

UI:            ui_{category}_{name}.png
               ui_window_corner_tl.png

アイテム:      item_{id}_{name}.png
               item_001_herb.png

エフェクト:    fx_{name}_{frame}.png
               fx_fire_01.png
```

## 6. Aseprite 設定メモ

- Grid Size: 8×8
- Pixel Aspect Ratio: 1:1
- Color Mode: Indexed Color
- パレット: マスターパレット.pal をインポートして使用
- エクスポート: PNG, Scale 1x, Trim: OFF
