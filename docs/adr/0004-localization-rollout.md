# ADR-0004 Localization Rollout

- Status: Accepted
- Date: 2026-03-15

## Context

MVP は 5世界 / 30体、Initial Release は日本語先行で進める方針になったが、日英同時対応を維持するとテキスト制作、レビュー、QA の負荷が本番出荷のボトルネックになりやすい。

## Decision

- Initial Release は日本語先行で出荷する
- 英語対応は後続フェーズで追加する
- テキスト管理構造は最初から多言語拡張可能にしておく

## Rationale

- まず日本語版の世界観、会話、伏線、図鑑文の密度を高める方が作品価値に直結する
- ソロ開発では翻訳レビューとローカライズQAが初回出荷の速度を大きく落としやすい
- データ構造だけ先に多言語対応にしておけば、後から英語を安全に足せる

## Consequences

- Initial Release の QA では日本語完備を必須とする
- 英語文字列の必須チェックはローカライズ実装フェーズまで持ち越す
- フォント、ID体系、データ構造は英語追加を見越して維持する
