# ADR-0006 Initial Release Story Completeness

- Status: Accepted
- Date: 2026-03-15

## Context

本作は多層構造の世界観と裏ストーリーを持つが、初回リリースが未完のまま終わると作品全体の印象が弱くなる。

## Decision

- Initial Release 時点でメインストーリーは完結させる
- 裏ストーリーは本編の不足分を補うものではなく、完結した本編の深層を開く追加層として設計する

## Rationale

- プレイヤーは初回リリースだけで一本の作品として満足できる必要がある
- その上で、裏ストーリーやエンドコンテンツによって「終わったと思ったらまだある」を成立させる方が強い
- 物語の骨格を初回で閉じておくと、後続の追加要素も未完感ではなく拡張として受け止められる

## Consequences

- メインストーリーの締めは Initial Release までに完成必須
- 深層設定や追加解釈はエンドコンテンツに回してよい
- クリフハンガー依存の構成は避ける
