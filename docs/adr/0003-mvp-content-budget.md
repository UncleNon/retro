# ADR-0003 MVP Content Budget

- Status: Accepted
- Date: 2026-03-15

## Context

Vertical Slice の次に置く中間マイルストーンの具体的なコンテンツ量を固定しないと、設計と実装の見積もりがぶれる。

## Decision

- MVP の暫定ターゲットを 5世界 / 30体 とする
- North Star は 20+世界 / 400体 のまま維持する
- MVP 用の世界設計、モンスター設計、テキスト見積もりは 5世界 / 30体 を基準に行う

## Rationale

- 5世界あれば、異なる文化・生態系・禁忌を見せながら、一本の作品としての変化を出しやすい
- 30体あれば、序盤から中盤にかけて配合・育成・愛着設計を成立させる最小限の厚みを持たせやすい
- それ以上を初回確約すると、アセット、テキスト、QA、データ設計が同時に膨張しやすい

## Consequences

- MVP の設計では、まず 5世界 / 30体 の完成度を最優先する
- 追加世界と追加モンスターは、後続フェーズで段階的に足す
- Vertical Slice 後に必要なら見直すが、現時点ではこの値を planning baseline とする
