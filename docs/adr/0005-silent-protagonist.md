# ADR-0005 Silent Protagonist

- Status: Accepted
- Date: 2026-03-15

## Context

主人公は45歳のおっさん/おばさんという強い設定を持つが、会話量を増やすとプレイヤー投影とテキスト制作コストの両面で重くなりやすい。

## Decision

- 主人公は無言主人公とする
- 意思表示は選択肢、短い反応、表情差分、演出、NPC側の反応で表現する

## Rationale

- プレイヤーが主人公に自分の解釈を乗せやすい
- 45歳主人公の重みは、台詞量ではなく背景、態度、周囲との関係で出せる
- 初回リリースを日本語先行・高密度でまとめる上でも、テキスト量を抑えつつ雰囲気を濃くしやすい

## Consequences

- 会話文は NPC とナレーション側で情報を運ぶ比重が高くなる
- 主人公の人格は選択肢と行動で示す
- 将来のローカライズ対象テキスト量も抑えやすくなる
