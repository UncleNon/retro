## 1. Act V Reveal Contract

Act V は `真相を知る` で終わらず、`誰が運用し、誰が得し、何を終わらせるかを選ぶ` 幕である。
`W-020` で中央の正統化装置を可視化し、`W-021` でその装置が依拠していた `名 / 所属 / 記録` の定義自体をほどく。

| 世界 | 受け持つ証明 | 主 clue |
|------|--------------|---------|
| `W-020` | 配合文化と失踪処理は中央で正統化されていた | `CL-050`〜`CL-052` |
| `W-021` | 塔は人格でなく歪みに反応し、主人公は沈黙を継がない選択を取る | `CL-053`, `CL-054` |

### 1.1 Act V の差分原則

| world_id | 役割 | 敵の怖さ | 人間臭い争点 | 最終的に問うこと |
|----------|------|----------|--------------|------------------|
| `W-020` | 運用責任の中心を見せる | 正統、記録、審問が一体で押し寄せる | 何を `正しい歴史` として残すか | 公的沈黙を壊せるか |
| `W-021` | 定義の外側で最後の選択をさせる | 名も所属も役に立たない | それでも誰を誰として扱うか | 記録を開く / 継がない意思を持てるか |

## 3. `W-021` ほどけの縁

### 3.1 Proof Contract

| 項目 | 内容 |
|------|------|
| premise | 名も所属も持ち込めない白泥の境界世界で、足跡と残響だけが `誰かだった痕` として残る |
| function | `terminal_world` |
| proof | 塔が反応していたのは人格でも神意でもなく、`名 / 所属 / 記録 / 身体` の不一致そのものだった |
| corridor / chain | `COR-05 fracture-descent` / `ECH-04 fracture-identity` |
| bloc / front | `例外領域` / すべての勢力の観測残滓 |
| clue | `CL-053`, `CL-054` |
| evidence | `白泥に残る、本人名と一致しない足跡列` |

### 3.2 Camp Topography

| camp_id | 仮設営地 | 主体 | 何を求めているか |
|---------|----------|------|------------------|
| `C-01` | `白杭観測列` | FAC-02 | 歪み反応を数字で確定したい |
| `C-02` | `帳棚仮市` | FAC-03 | ほどける前の所属断片を押さえたい |
| `C-03` | `灰幕営` | FAC-04 | 死でも帰還でもない者の待ち方を主張したい |
| `C-04` | `骨車泊` | FAC-06 | 遺留品と白泥試料を高値で抜きたい |
| `C-05` | `家印小幕` | FAC-01, FAC-07 | 家の名誉と相続権の証拠を取り戻したい |

各 camp は `停泊地 / 観測列 / 待機幕 / 帳棚` の仮設群であり、
`worlds/05_world_catalog_and_budget.md` の settlement 1 は
全 camp を束ねる前営 `白縁前営` を指す。

### 3.3 Taboo / Axes

| 項目 | 内容 |
|------|------|
| taboo | 白縁の内側で、自分の名を確定した言い方で呼ぶな |
| actual operation | どの camp も `仮札`, `位置名`, `役割名` で人を呼ぶしかなく、持ち込んだ帳面や印は短時間で意味を失う |
| silence circuit | 観測係、 camp 書記、遺族待機者、境界兵、 sample 運び、偽解釈 broker |
| shortcut market | 一時仮札、白泥 sample の売買、 `自説に都合のよい足跡解釈`, 亡失帳面の回収代行 |
| `Power` | 主権はなく、各勢力が camp 単位で interpretation war をしている |
| `Faith` | 清めの極地と見る者も、塔の底が露出した場所と見る者もいる |
| `Work` | 足跡記録、白泥封入、残響採取、 temporary route づくり、行方不明 camp の維持 |
| `Blood` | 血縁申立ては通らず、ここでは `誰を迎えに来たか` の意志だけが最後に残る |
| `Monster` | divine, magic, dragon は `world law のほつれ` として現れ、従来の系統規則が半分しか通じない |
| `Boundary` | 所属印、 badge, 家名布、通行札はすべて参考程度に落ち、続柄や位置でしか人を指せない |

## 4. Act V 共通メモ

- `W-020` では `制度の中心` を、`W-021` では `制度が依拠していた定義の限界` を見せる
- `W-020` は `誰が得してきたか` の集積、`W-021` は `それでも何を人として扱うか` の選択に寄せる
- ここでは新しい謎を増やさず、既出の `ECH-01〜04` と `CL-001〜054` を閉じる側へ使う
- 終盤ボスは `強い` より `これまで学んだ語彙が全部試される` を優先する

## 5. QA Checklist

- `W-020` が単なる大聖堂ダンジョンでなく、中央運用の実務になっているか
- `W-021` が抽象空間に逃げず、 camp / 遺族 / 書記 / 現場官僚の生活痕を持っているか
- 終盤で本編外の宇宙論へ逃げず、共同体と制度の責任で閉じているか
- ending choice が善悪二択でなく、何を残し何を壊すかの選択になっているか
