# 04. Dungeon Template Catalog

> **ステータス**: Draft v1.0
> **最終更新**: 2026-03-15

---

## 1. 目的

- 20+世界でダンジョンが単調化しないよう、型を先に定義する
- ランダム生成と固定マップの使い分けを整理する

---

## 2. ダンジョンの大分類

| 分類 | 主用途 | 向いている実装 |
|------|--------|----------------|
| 生活圏型 | 村や町の外れ、採集、墓地 | 固定 |
| 儀式型 | 塔、門、神殿、祭場 | 固定 |
| 生態型 | 洞窟、巣、湿地、根脈 | 固定 + 一部可変 |
| 制度型 | 監獄、記録庫、役所地下、競技施設裏 | 固定 |
| 深層型 | 異界、門内部、世界の継ぎ目 | ハイブリッド |
| 周回型 | 無限回廊、試練塔 | ランダム中心 |

---

## 3. テンプレート一覧

### T-01 一本道圧迫型

- 用途:
  - 塔への導線
  - 不安を高める前口上
- 特徴:
  - 一本道
  - 視認できる横道はあるが行けない
  - ambient が減る

### T-02 生活痕探索型

- 用途:
  - 村外れの納屋、墓地、畑、物置
- 特徴:
  - 戦闘より調べ物
  - 記録物、札、家印、工具を置ける

### T-03 群れ圧型

- 用途:
  - 巣、鳥塚、湿地、獣穴
- 特徴:
  - 同系統の敵が連続で出る
  - 状態異常や囲まれ感を出しやすい

### T-04 記録分断型

- 用途:
  - 記録庫、役所裏、寺院書庫
- 特徴:
  - 文書がバラバラに配置
  - 順路を追うと制度の隠蔽が見える

### T-05 儀式同心円型

- 用途:
  - 門前祭場、塔内部、封印室
- 特徴:
  - 中央に向かって寄る
  - 周辺の装飾が反復
  - 一本道より“儀式感”が強い

### T-06 上下反転型

- 用途:
  - 鏡世界、夢世界、門内部
- 特徴:
  - 同じ部屋が微妙に違う
  - 遠近感を狂わせやすい

### T-07 生活と死の重なり型

- 用途:
  - 共同墓地、葬具庫、廃畜舎
- 特徴:
  - 生活道具と弔い道具が混在
  - ダークさを強くできる

### T-08 周回試練型

- 用途:
  - 試練塔、裏ダンジョン
- 特徴:
  - 同一ルールの変奏
  - フロア修飾子と深度で差を出す

## 4. Vertical Slice で使う型

| 対象 | テンプレート |
|------|-------------|
| 塔前荒地 | T-01 一本道圧迫型 |
| 記録小屋 | T-02 生活痕探索型 |
| 塔内部前半 | T-05 儀式同心円型 |
| 最初の越境先小ダンジョン | T-03 群れ圧型 + T-04 記録分断型 |

## 5. ダンジョンごとの必須要素

| 要素 | 最低数 |
|------|-------:|
| 固有オブジェクト | 3 |
| 固有会話 or 記録物 | 2 |
| encounter table の個性 | 1 |
| 隠し導線 | 1 |
| 伏線 | 1 |
| 通常時と深層時の空気差 | 1 |

---

## 6. ギミック使用ルール

### 序盤で使ってよいもの

- 一本道
- 簡易ワープ
- 隠し通路
- 暗がり
- 滑る床の軽量版

### 中盤以降に回すもの

- 複数層連動スイッチ
- 時間差ギミック
- 所持モンスター系統で開く道
- 記録改竄を読む必要がある謎解き

### 終盤向け

- 正しい名前 / 所属 / 記録を揃えることで開く門
- 配合履歴や変異録を参照する深層ギミック

## 7. 尺の目安

| 種別 | 目安プレイ時間 |
|------|---------------:|
| 生活圏型 | 3〜8分 |
| 小型ダンジョン | 8〜15分 |
| 中型ダンジョン | 15〜25分 |
| 世界メインダンジョン | 25〜40分 |
| 裏ダンジョン | 30分〜 |

---

## 8. 禁止事項

- ただ長いだけの通路
- トゲ床や落とし穴の連打で時間を稼ぐ
- 手応えよりストレスが勝つ視界制限
- 文脈のないパズル
