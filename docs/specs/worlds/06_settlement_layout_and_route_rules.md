## 1. 目的

- 20+世界を作っても、マップ品質と探索テンポが崩れないようにする
- 村、町、道、野外、塔前、祠、ダンジョン入口の数値テンプレを固定する
- 「雰囲気で広さを決める」作り方をやめ、タイル単位で内容量を制御する

## 2. 基本規格

| 項目 | 値 |
|------|----|
| 内部解像度 | `160 x 144` |
| 基本タイル | `8 x 8` |
| 1画面 | `20 x 18 tiles` |

## 4. 村と町の導線設計

### 4.1 導線の基本

- 入口から `最初に目につく施設` を1つ置く
- 主要施設は `三角導線` を作る
- 一本道の直線集落は避け、必ず `視界外の小脇` を1箇所以上作る

### 4.2 三角導線

| 施設A | 施設B | 施設C |
|-------|-------|-------|
| 休息拠点 | 道具屋 | その町固有の施設 |

### 4.3 理想距離

| 区間 | タイル数 |
|------|---------:|
| 入口→宿 / 休息 | 10〜18 |
| 入口→道具屋 | 12〜24 |
| 宿→道具屋 | 8〜16 |
| 道具屋→牧場 | 10〜20 |
| 村中央→外周の禁忌地点 | 16〜36 |

## 5. 開始村テンプレ

### 5.1 必須構成

| 要素 | 条件 |
|------|------|
| 主人公宅 | 村の周辺寄り、日常の起点 |
| 家畜小屋 / 囲い | 主人公職能を示す |
| 井戸 / 水場 | 生活感の中心 |
| 長老宅 or まとめ役の家 | 共同体の沈黙の核 |
| 小祠 / 空碑 / 供養物 | 禁忌の痕 |
| 塔が見える地点 | 1画面以上から視認可能 |

### 5.2 禁止

- チュートリアル専用の無機質な広場
- 生活感のない記号的な家並み
- 塔が全く視界に入らない開始村

## 6. 野外フィールドテンプレ

### 6.1 サイズ帯

| 種別 | tile size 目安 | 説明 |
|------|----------------|------|
| `approach` | `48x32` 〜 `72x40` | 拠点の外、入口演出用 |
| `field_small` | `64x48` 〜 `80x56` | 序盤の短距離フィールド |
| `field_medium` | `80x56` 〜 `112x72` | 中盤の基幹野外 |
| `field_large` | `112x72` 〜 `144x96` | 一本道でなく多方向探索を持つ |

### 6.2 ランドマーク密度

| field type | ランドマーク数 |
|------------|---------------:|
| `approach` | 1〜2 |
| `field_small` | 2〜4 |
| `field_medium` | 4〜6 |
| `field_large` | 6〜9 |

### 6.3 入口数

| field type | 入口 / 出口数 |
|------------|---------------:|
| `approach` | 1〜2 |
| `field_small` | 2〜3 |
| `field_medium` | 3〜4 |
| `field_large` | 4〜6 |

### 6.4 遭遇設計

- 一本道では `terrain_rate 0.90〜1.00`
- 寄り道の高草地では `terrain_rate 1.10〜1.25`
- 危険な最短路より、`安全な遠回り / 危険な近道` の択を優先する

## 7. 塔前 / 門前テンプレ

### 7.1 役割

- その世界の禁忌を `最初の一枚` で見せる
- 普通の世界と異物の境界を視覚で理解させる
- NPC台詞なしでも「ここは近づくべきでない」と分からせる

### 7.2 数値テンプレ

| 項目 | 値 |
|------|----|
| map size | `48x32` 〜 `72x48` |
| 視認距離 | 入口から塔本体が1回は画面内に入る |
| 会話NPC | 0〜2 |
| 戦闘回数目安 | 0〜3 |
| 固定イベント物 | 2〜4 |

### 7.3 必須オブジェクト

- 境界杭 or 柵
- 供物 or 捨て置かれた生活物
- 文字の削れた看板 or 碑
- 不自然な静音領域 or 風向き差

## 8. 祠 / 小施設テンプレ

### 8.1 用途

- ボス前の圧縮空間
- 祈り、供養、記録、封印の局所表現
- 1テーマだけ濃く出す

### 8.2 サイズ

| 種別 | tile size |
|------|-----------|
| 小祠 | `24x24` 〜 `40x32` |
| 中祠 | `40x32` 〜 `56x40` |
| 儀礼施設 | `56x40` 〜 `72x48` |

### 8.3 ルール

- 小さい分、視線誘導は強くする
- 一本道にしすぎず、短い迂回を1つ入れる
- イベント物は `3つまで` に絞る

  - 中世界: 11

## 12. QA Checklist

### 12.1 Settlement

- 入口から目的施設への距離が遠すぎないか
- 生活感と禁忌の痕が両立しているか
- どの家も同じ形に見えないか
- 主人公職能と町の産業が噛んでいるか

### 12.2 Field

- 一本道で終わっていないか
- 安全路 / 危険路 / 寄り道が分かれているか
- ランドマークが弱すぎないか
- エンカウント密度がマップ長に見合っているか

### 12.3 World identity

- その世界の禁忌が形で見えるか
- 塔や門と同じ言語で作られているか
- 他世界と地形、制度、生活物が被りすぎていないか
