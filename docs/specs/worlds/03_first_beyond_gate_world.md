## 1. 位置づけ

最初の門の向こう側の世界は、プレイヤーに次の 3 つを確信させるための最初の証拠世界である。

1. 塔の禁忌は故郷の村だけの迷信ではない
2. `名前 / 家印 / 所属 / 家畜管理` は世界をまたいで同じ歪み方をしている
3. モンスターの奇妙さは、自然現象だけでなく人間の制度や記録の歪みとも結びついている

この世界は `壮大さ` よりも `変奏` を優先する。開始村とよく似た生活の手触りを持ちながら、禁忌の言い換え方と記録の扱いが一段露骨である。

## 3. この世界の禁忌

### 3.1 表向きの説明

- 日が落ちた後に本名を呼ぶと、風がそれを持っていく
- 柵の外で真名を呼ばれた者は、帰る道を見失う
- 生まれたばかりのもの、拾われたもの、よそ者には **借り名の札** を与える

### 3.2 実際の運用

- 村人は日常的に役割名で呼び合う
- 家ごとに色の違う札と鈴があり、所属を目で管理する
- 借り名のまま一定期間を過ぎると、公式記録から本名が薄れていく

### 3.3 主人公の故郷との接続

| 故郷の村 | 名伏せの野 |
|----------|------------|
| 塔に近づくな | 外で真名を呼ぶな |
| 家畜札の削り跡 | 借り名札の上書き |
| 表札だけ外された家 | 本名を書かない戸口 |
| 空碑 | 名前を刻めない供養板 |

## 4. 世界の役割

### 4.1 ストーリー上の役目

- 主人公は「自分の村だけが異常だったわけではない」と理解する
- 失踪と改名が人為的に運用されている制度の可能性が見え始める
- 最初の門が単なる移動装置でなく、`所属を数え直す何か` だという印象を強める

### 4.2 システム上の役目

- 勧誘補助、睡眠、毒、印操作を試せる
- `marked` と `recruit` の連携を自然に学ばせる
- 初めて `壁 / 回復 / 封印` が必要になるボス構造を置く

## 6. 地形と生態

### 6.1 地形レイヤ

| レイヤ | 内容 |
|--------|------|
| 高地 | 乾いた灰草、木柵、風見鈴 |
| 中地 | 放牧地、札杭、踏み固められた道 |
| 低地 | 湿った葦原、名札の流れ着く浅水域 |
| 聖域 / 祠 | 柵が内側に向いている、石ではなく白い骨質の柱 |

### 6.2 生態ルール

- ここではモンスターも人も `鳴き声 / 呼び声 / 印` に敏感
- 鳥系は鈴に寄り、植物系は濡れた札に寄る
- 家畜系モンスターは柵の境界に沿って歩き、真ん中に出たがらない
- 風が止む瞬間だけ、名前を呼ぶ声に似た雑音が混じる

### 6.3 環境ハザード

| ゾーン | ハザード | 効果 |
|--------|----------|------|
| 放牧地 | 高い草 | エンカウント `terrain_rate 1.0` |
| 葦原 | 湿名の泥 | `terrain_rate 1.25`、`wet` 付与イベントあり |
| 裏柵の祠 | 狭路 | 逃走率 `-10` |

- 故郷の村との一致はプレイヤーが察する前提で、直接比較は終盤まで避ける

## 9. ハブ: 継札集落

### 9.1 外観

- 柵と風除け布に囲まれた小集落
- 家の表札はなく、代わりに色違いの札束が下がっている
- 戸口ごとに鈴の音色が違う

### 9.2 主要施設

| 施設 | 役割 |
|------|------|
| 継札小屋 | 借り名札の管理 |
| 鈴結い小屋 | 鈴と札の修繕 |
| 乾草棚 | 一時セーブ / 休息拠点 |
| 柵守詰所 | 裏柵の祠への導線 |

### 9.3 プレイヤー体験

- 最初は「妙な文化だが生活は回っている」と見える
- 話を聞くほど、名前が文化でなく **管理手段** に近いと分かる

## 10. メインルート

### 10.1 進行順

1. 継札の門前で世界の音と風の違いを知る
2. 継札集落で借り名文化の説明を受ける
3. 放牧地で `marked` と勧誘補助の基本を体験
4. 湿名の葦原で `sleep / poison / wet` を学ぶ
5. 裏柵の祠でボス `逆杭のマヨイカカシ` と対決
6. 返名の中庭で帰還手段を開く

### 10.2 中間目標

| 目標 | 内容 |
|------|------|
| `OBJ-W01-001` | 継札番から裏柵の事情を聞く |
| `OBJ-W01-002` | 放牧地で `borrowed bell tag` を回収 |
| `OBJ-W01-003` | 葦原で濡れた札の束を見つける |
| `OBJ-W01-004` | 祠で裏向きの柵の理由を知る |

## 11. ボス仕様

### 11.1 `逆杭のマヨイカカシ`

| 項目 | 値 |
|------|----|
| 系統 | material |
| 想定ランク | D+ |
| 想定Lv | 14 |
| 主要スキル | `まよわせる`, `しるしうばい`, `みみざわりの鈴`, `にぶいひかり` |
| ギミック | `marked` を奪って自分の火力に変える |
| ボス意図 | プレイヤーに `みやぶる` の便利さと危険さを同時に教える |

### 11.2 戦闘設計

- 開幕でプレイヤー 1 体を `marked`
- `しるしうばい` で回収
- `みみざわりの鈴` で `hushed` を撒き、補助の再使用を遅らせる
- 対抗手段として `やわらかいかべ`, `くさのしずく`, `ふういん` が刺さる

## 12. 演出ルール

### 12.1 音

| 場所 | ルール |
|------|--------|
| 門前 | 風 + 遠い鈴。BGM は薄い |
| 集落 | 家ごとに鈴音が違う |
| 葦原 | 風が止むたびに囁きに似た帯域が混ざる |
| 祠 | 鈴が鳴らず、代わりに札が擦れる音だけが残る |

### 12.2 色

- 故郷の村より彩度は低い
- ただし死んだ世界にはしない。草、布、札にだけ彩りを残す
- 湿地は青緑、柵は灰白、札はくすんだ赭色

### 12.3 UI

- この世界では NPC 名表示を役割名中心にする
- 何人かは固有名がログに出ない
- プレイヤーが違和感を覚えるが、チュートリアルで説明しない

## 13. 伏線と回収ポイント

| 要素 | 故郷との接続 | 後の回収 |
|------|--------------|----------|
| 借り名札 | 家畜札の削り跡 | 名前と所属の改竄 |
| 表札の不在 | 表札だけ外された家 | 所属が家単位で上書きされる制度 |
| 空の供養板 | 空碑 | 死として処理されない失踪 |
| 外で真名を呼ばない | 塔に近づくな | 門が名前に反応する仕組み |
| 裏向きの柵 | 北道の柵 | 守る対象は外ではなく内側 |

## 14. 実装メモ

- 初回世界として広すぎないが、`別世界に来た` 感は十分に出す
- 故郷との類似は配置と制度で見せ、見た目を単純コピーしない
- 会話は `説明口調` より `運用口調` を優先する
- この世界で真相を言い切らない。プレイヤーに「制度の形が似すぎている」と思わせれば十分
