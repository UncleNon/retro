## 1. 目的

- 開始村の外観、寸法、導線、塔との距離感を先に固定する
- 「生活感」と「異物としての塔」の対比を、実装しやすい数値で定義する

村名はまだ仮称とし、この文書では **開始村** と表記する。

## 8. 塔の見せ方

### 可視距離

- 村中央からは塔全体を見せない
- 北道に入ると上部 20〜30% が見える
- 塔前荒地で初めて全景を見せる

### 塔スプライト / 建築寸法（外観用）

| 項目 | 値 |
|------|----|
| 可視外観幅 | `18 tiles` |
| 可視外観高 | `28 tiles` |
| 入口幅 | `4 tiles` |
| 前庭奥行き | `10 tiles` |

### 演出ルール

- 塔周辺だけ地面の色温度を 1段冷たくする
- BGM は切らず、環境音を薄くする
- 家畜 / 鳥系の ambient を消す
- 一本道にして、寄り道を意図的に減らす

## 10. 実装メモ

- Tiled 上では `96 x 64` の1マップで管理し、室内は別 map に切る
- `Object Layer` に `npc_spawn`, `inspect_point`, `event_trigger`, `encounter_zone` を置く
- 北道は **幅6タイル** を下回らないようにし、窮屈すぎる見た目を避ける
- 村の道は `2〜4タイル幅`、広場は `10 x 8タイル以上` を確保する
