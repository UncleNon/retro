## 3. モンスター1体の Definition Of Done

### デザイン

- 名前（日/英）
- 系統
- ランク
- モチーフ群
- シルエット種別
- サイズ
- パレットID
- 図鑑トーン

### 数値

- `base_level_cap`
- `growth_curve_id`
- `base/cap` の7ステータス
- `base_recruit`
- 耐性
- trait 2つ
- innate skill 3つ

### 入手導線

- 勧誘可否
- 出現ゾーン
- 出現重み
- 出現Lv帯
- 配合素材としての役割

### アート

- battle sprite
- field sprite
- 顔 / 向き差分要否
- 最低2フレームアニメ
- prompt
- negative prompt
- 手修正メモ

### 世界観

- 図鑑文 1本
- 生態メモ
- 禁忌 / 伏線との接続点 1つ以上

### QA

- 既存IP類似チェック
- battle / field で視認性確認
- 配合履歴表示確認

---

## 4. 世界1つの Definition Of Done

### 世界定義

- 世界名（日/英）
- テーマ
- 政治
- 信仰
- 禁忌
- 経済
- dark secret

### ゲームプレイ

- レベル帯
- 出現系統
- 固有モンスター数
- 拠点数
- ダンジョン数
- gate 解放条件
- 固定イベント数

### 視覚

- タイルセット
- 外観色ルール
- 建材ルール
- ambient / BGM の方向性

### ストーリー

- その世界の伏線 3つ以上
- その世界の失踪 / 改名 / 所属揺らぎの変奏
- 主要NPC 3人以上
- 本編での役割

---

## 5. NPC1人の Definition Of Done

- 名前
- 年齢帯
- 表向きの役割
- 裏の役割
- 配置座標
- 会話回数 / 会話段階
- 初回台詞
- 条件分岐台詞
- 伏線担務
- 所持モンスター（必要時）
- 移動パターン

---

## 6. スキル1つの Definition Of Done

- `skill_id`
- 名前（日/英）
- 種別
- 属性
- MPコスト
- 対象範囲
- base power / base rate
- 成否演出
- AI評価タグ
- 説明帯テキスト

---

## 7. アイテム1つの Definition Of Done

- `item_id`
- 名前（日/英）
- 種別
- 価格
- 売値
- 使用可否
- 使用対象
- 効果式
- 説明帯テキスト
- ドロップ / 店売り / 生成元

---

## 8. ダンジョン1つの Definition Of Done

- マップサイズ
- 階層数
- gimmick 2つ以上
- encounter table
- 固定宝箱
- 隠し導線
- ボス条件
- 入口と出口の意味
- BGM / ambient
- 伏線 1つ以上
