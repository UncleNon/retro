# 00. Master Design Matrix

> **ステータス**: Draft v1.0
> **最終更新**: 2026-03-15
> **目的**: 「まだ決めることが大量にある」ことを前提に、設計面を漏れなく棚卸しする

---

## 1. 使い方

- この文書は「何を決める必要があるか」の母表
- `fixed`: 現時点で大枠を固定済み
- `next`: 早めに詳細化すべき
- `open`: まだ深掘りが必要
