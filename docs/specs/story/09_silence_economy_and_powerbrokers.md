## 1. 目的

- 各 faction を `理念` でなく `日銭と保身` で動く存在として具体化する
- どの世界で誰が口止めをし、誰が利ざやを取り、誰が一番先に切られるかを固定する
- main story の中盤以降で、勢力衝突が会話だけでなく `現地の運用差` として見えるようにする

## 2. 沈黙経済の五層

| 層 | 誰がいるか | 何をしているか | 何で儲けるか |
|----|------------|----------------|--------------|
| `L1 家内` | 家長、未亡人、養親、長男長女 | 名を残すか消すか決める | 相続維持、再婚保留、家の体面 |
| `L2 現場` | 墓守、記録番、港吏、柵守、門守 | case を分類し、現場を整える | 手数料、見逃し賃、優先枠 |
| `L3 地方中継` | 代書人、巡礼先導、宿場主、商長補佐 | 情報と人を次の制度へ流す | 斡旋料、宿銭、運賃差益 |
| `L4 中央整合` | 法官、上席書記、聖堂整理役、監査官 | 食い違う記録を `一応合う` ようにする | 税、認証料、昇進、保身 |
| `L5 例外処理` | 密輸頭、封鎖実務監、裏市場仲介 | 制度外のものを処理して隠す | 禁制品利幅、事故口止め金 |

## 3. faction ごとの運用実態

### 3.1 FAC-01 村落長老ネットワーク

| 項目 | 内容 |
|------|------|
| 表向きの任務 | 村の平穏、冬越し、祭り、葬送の調停 |
| 実際の収入 | 紛争仲裁の礼、婚姻取りまとめ、季節雇いの口利き |
| 隠し資産 | 借り名帳、拾い子の実親線、墓所の空区画 |
| 得意な隠蔽 | 人名を家名に溶かす、口伝だけ残す |
| 一番切り捨てる相手 | 継子、薄印の子、よそから来た嫁 |

主要 middle actor:

- **ヨノマ**: 牧印諸環の婚姻仲立ち婆。誰が誰の子かを知るが、寒冬の年ほど嘘を高く売る。
- **ガンセ**: 北道の古杭守。事故後に杭の文言を書き換える実務を請け負う。
- **ミホロ**: 空碑まわりの供物番。遺族へ優しいが、弔いを閉じないことで香代を取り続ける。

### 3.2 FAC-02 塔保全機構

| 項目 | 内容 |
|------|------|
| 表向きの任務 | 門事故の防止、封鎖、観測、危険通行の管理 |
| 実際の収入 | 通行許可、封鎖例外、反応測定の予算 |
| 隠し資産 | 反応ログ、失格者名簿、封鎖前後の改訂命令 |
| 得意な隠蔽 | `事故` を `規定外通行` に言い換える |
| 一番切り捨てる相手 | 現場門守、下級監視員、同行家族 |

主要 middle actor:

- **トウセイ**: 岬側の封鎖主任。 clean pass の発行枠を握り、事故を潮のせいにする。
- **イオリ**: 閉鐘都の実務監。良心が残るが、現場を守るために報告数を削る。
- **マシラ**: 鏡森調査隊の記録兵。別所属名の映りを見た者から先に遠方転属させる。

### 3.3 FAC-03 記録官連盟

| 項目 | 内容 |
|------|------|
| 表向きの任務 | 戸籍、通行、家系、労働、相続の整合管理 |
| 実際の収入 | 抄本料、再発行料、照会料、矛盾調停料 |
| 隠し資産 | 複本台帳、後補記簿、廃棄前写し |
| 得意な隠蔽 | 転出、奉公替え、名寄せエラーとして消す |
| 一番切り捨てる相手 | 無籍労働者、未完婚礼者、遠隔地の出稼ぎ |

主要 middle actor:

- **スズネ**: 二署法台の若書記。昇進のため黒簿を抱え続けている。
- **ハサミ**: 塩庫の耐塩紙仲買。濡れた台帳だけを選んで買い、後補記材料へ回す。
- **ネブロ**: 宿場の写本師。客の偽名を清書するのがうまく、失踪の最初の言い換え役になる。

### 3.4 FAC-04 帰還なき祈祷会群

| 項目 | 内容 |
|------|------|
| 表向きの任務 | 弔い、巡礼、魂鎮め、未帰還者への慰撫 |
| 実際の収入 | 供物代、巡礼税、代理祈祷、 badge 販売 |
| 隠し資産 | 未完葬送簿、逆誓詞注釈、空碑対応記録 |
| 得意な隠蔽 | 神意、待つべき者、還る途上として濁す |
| 一番切り捨てる相手 | 金のない遺族、顔を晒したい家族、教義に逆らう下僧 |

主要 middle actor:

- **ツグラ**: 黒布巡礼団の先導。姉を死者扱いしたくない私心から教義をねじる。
- **ソヨ**: 香なし墓苑の供香売り。香を焚かない方が戻らぬと説きながら、裏で安香も売る。
- **レンカ**: 逆誓の修院注釈係。順唱へ戻す派に内通している。

### 3.5 FAC-05 闘技会運営局

| 項目 | 内容 |
|------|------|
| 表向きの任務 | 門競、資格認定、観戦経済の統括 |
| 実際の収入 | 興行、賭け、 rank 認定、スポンサー献納 |
| 隠し資産 | 失格者処理簿、適応者観測一覧 |
| 得意な隠蔽 | 自己都合離脱、規約違反処理 |
| 一番切り捨てる相手 | 若年参加者、無名出場者、地方推薦枠 |

主要 middle actor:

- **サエモ系譜のアラタ**: 初代総裁の名声を食う興行主任。事故より視聴率を取る。
- **トガリ**: 岬灯海路盟と組む賭場締め。港と門都の客を同じ帳面で回す。

### 3.6 FAC-06 門越え密輸網

| 項目 | 内容 |
|------|------|
| 表向きの任務 | なし。表では荷運びや仲買として散る |
| 実際の収入 | 偽印、骨荷混載、 clean pass 偽造、禁制素材 |
| 隠し資産 | 港の裏桟橋、荷札型、封鎖予定の先情報 |
| 得意な隠蔽 | 正規荷に紛れ込ませる、事故前に抜く |
| 一番切り捨てる相手 | 雇われ船員、荷役の日雇い、末端の badge 売り |

主要 middle actor:

- **ハクヤ**: 骨樋港の商長補佐。感情より運賃表を信じる。
- **ミツロ**: clean pass 型を削る印屋。病より手数料を怖がる。
- **エダン**: 残名回廊へ遺留品を流す拾い屋。遺族へは `手掛かり` として売る。

### 3.7 FAC-07 家門連鎖同盟

| 項目 | 内容 |
|------|------|
| 表向きの任務 | 名家の継承安定、婚姻調整、家産保全 |
| 実際の収入 | 婚資、相続調停、系譜修復、後見料 |
| 隠し資産 | 分家墓列、養親線、薄印除外の覚書 |
| 得意な隠蔽 | 正統とされる枝だけを残す |
| 一番切り捨てる相手 | 第二夫人の子、継子、修復家系の寡婦 |

主要 middle actor:

- **フユネ**: 炉雪諸家の婚資監。冬名を売り物にし、夏名の人間を二級扱いする。
- **ツギハ筋のサラギ**: 焼継ぎ戸籍の調停屋。修繕証と偽家督証の境目を曖昧にして生きる。

## 6. 本編で効く人間臭い争点

1. 正義より `今ここで家が潰れるか` が優先される
2. 真相より `未亡人が再婚できるか` が争点になる
3. 上位機関の嘘より `末端が少しずつ楽をした積み重ね` が重い
4. 善人ほど、自分が助けた数人のために全体の嘘を延命させる
5. 若手は改革を語るが、大体は家の負債か昇進に追われている

## 8. QA Checklist

- faction が `善悪ラベル` でなく `業務と都合` で見えるか
- 金の流れと証拠隠しの流れが一致しているか
- 各 faction に `少しは本当に守ったもの` があるか
- 主要人物が全員 `中央の悪人` になっていないか
- 本編で露出する争点が、家族と生活へ降りてくるか
