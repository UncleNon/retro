## 1. 文書目的

この文書は、以下を一本化する。

1. 本作の物語が何を描く作品なのか
2. 世界がどういう法則で成り立っているのか
3. 主人公が何を知り、何を誤解し、何を選び取るのか
4. プレイヤーに何をいつ教え、何を残すのか
5. Initial Release で何を完結させ、何を postgame deep lore に送るのか

本作の物語は、`雰囲気先行のダークファンタジー` ではなく、`共同体・制度・継承・名付け・配合・世界越境` が一つの論理で繋がる構造物として設計する。

## 2. 作品の核

### 2.1 一文定義

> 小さな村で家畜番として生きてきた45歳の無言主人公が、昔から禁忌とされ、村人失踪の記憶がまとわりつく塔へ自ら踏み込み、人とモンスターと世界の「継ぎ足し」の歴史を暴き、共同体が何代にもわたり続けてきた沈黙の方法を終わらせる物語。

### 2.2 プレイヤーに最終的に残したい感情

| 項目 | 目標 |
|------|------|
| **序盤** | 懐かしいのに妙に生々しい |
| **中盤** | 世界が広がるほど同じ傷跡が見えてくる |
| **終盤** | 悪人を倒すだけでは終わらない責任がある |
| **クリア後** | 物語は完結したが、世界はまだ底が見えない |

### 2.3 中心テーマ

| テーマ | 定義 | 物語への落とし込み |
|--------|------|--------------------|
| **継承と断絶** | 名、家、技、血、所属、世界は継がれるが、継ぎ方を誤ると歪む | 配合、養子、相続、門、世界分裂が同じ構文で語られる |
| **管理された命** | 生き物を数える、札を付ける、所有する、登録する行為の暴力 | 家畜番主人公、戸籍、家畜札、闘技会登録、図鑑 |
| **共同体の沈黙** | 人々は悪意だけでなく生活防衛のために真実を隠す | 長老、記録官、宗教者、商人がそれぞれ別の理屈で沈黙を正当化する |
| **大人の冒険** | 若さの衝動ではなく、長年の違和感に責任を取る旅 | 主人公45歳、無言、大げさな英雄演出を避ける |
| **わからなさの維持** | 全部を解説しないことで世界の深さを保つ | 上位存在、原初世界、塔の意志は初回で断定しない |

### 2.4 避けるべき安易さ

1. 血筋だけで主人公を特別扱いしない
2. 黒幕一人で全問題を回収しない
3. 塔を科学装置や古代兵器に矮小化しない
4. ダークさを残酷描写の量で表現しない
5. postgame 用の謎を理由に本編を未完にしない

## 3. Canonical Truth Contract

この作品は「何が確定真実で、何を意図的に未確定に保つか」を契約として管理する。

### 3.1 Initial Release で真実として確定させるもの

| 項目 | 確定内容 |
|------|----------|
| 主人公の立場 | 塔の外の普通の生活圏に属する、45歳の家畜番 |
| 村の禁忌 | 迷信ではなく、実際の村人失踪に基づく共同体の記憶 |
| 失踪の共通項 | `名前 / 家系 / 所属` のいずれかに歪みがある者へ門が反応しやすい |
| 人間側の責任 | 複数勢力が歪みを知りつつ、記録・宗教・制度・物流で隠し続けてきた |
| 配合との接続 | 配合は単なる育成技術ではなく、世界の継ぎ足し法則を人間が利用したもの |
| 本編の結末 | 失踪を支えてきた人間社会の運用を止め、共同体の沈黙を破る |

### 3.2 Initial Release で断定しないもの

| 項目 | 理由 |
|------|------|
| 上位存在の正体 | 神、原初生命、世界自体の自己修復機構など、断定すると世界が小さくなる |
| 原初の単一世界の正確な成り立ち | postgame deep lore の主題に残す |
| 塔に人格や意思があるか | あるようにも見え、無いようにも見える余白が必要 |
| モンスターと人間の原初境界 | 図鑑・変異・残響で示すが、解答は残す |

### 3.3 Story QA 原則

- 本編だけで `事件`, `責任`, `選択`, `終止符` が成立していること
- postgame の役割は `補完` ではなく `深化`
- 主要謎は、最低 `3媒体 × 3回` で異なる角度から提示すること

## 4. 世界法則

### 4.1 四つの根

この世界の深層では、以下の四つが一本の法則で結ばれている。

1. **Name**: 何者として呼ばれるか
2. **Lineage**: どの流れから受け継いだか
3. **Affiliation**: どこに属すると記録されているか
4. **Gate**: 境界をまたぐための器官

### 4.2 用語定義

| 用語 | 定義 |
|------|------|
| **真名** | 儀礼・契約・弔いの場で効力を持つ名 |
| **通称** | 日常で使う名 |
| **家名** | 家門、屋号、家畜印、戸籍台帳に紐づく名 |
| **系譜** | 血縁だけでなく、養子、継子、婚姻、師弟、奉仕関係まで含む継承線 |
| **所属** | 家、寺院、商会、軍、闘技局、村落など、外部に証明される帰属 |
| **残響** | 失踪者が完全消滅せず、門の向こうの継ぎ目に残る痕跡 |
| **継ぎ足し** | 異なるものを結び、ずれごと成立させる行為。配合もその一形態 |

### 4.3 深層法則

| 法則ID | 名称 | 内容 |
|--------|------|------|
| L-01 | **名の固定** | 生き物は名によって世界内の位置を固定される |
| L-02 | **継承の重さ** | 継承は権利だけでなく負荷も受け継ぐ |
| L-03 | **所属の空白** | 所属が二重、空白、偽装になった者は境界に引っかかる |
| L-04 | **門の共鳴** | 門は人格ではなく、継ぎ目の不安定さに反応する |
| L-05 | **残響の保持** | 失踪は即死ではなく、継ぎ目側に痕跡が残りうる |
| L-06 | **配合の模倣** | 配合は門や継ぎ足しの法則を人間社会へ縮小実装したもの |

### 4.4 生活レベルでの迷信化

深層法則は、庶民の生活では以下のような形に変質して流通している。

| 深層法則 | 庶民レベルの言い回し |
|----------|----------------------|
| 名の固定 | 改名した者をすぐ門へ近づけるな |
| 継承の重さ | 継ぎ子を境目へ立たせるな |
| 所属の空白 | 帰る家のない者を塔へやるな |
| 門の共鳴 | 焼けた印章を持って塔へ行くな |
| 残響の保持 | 行方知れずの者の名を軽々しく呼ぶな |
| 配合の模倣 | 血の合わぬものを混ぜるな |

## 5. 塔と生きた門

### 5.1 塔の定義

- 塔は世界外の抽象空間ではない
- 山、谷、畑、牧草地、道、村、城がある通常のファンタジー地形の中に異物として立つ
- 外観は `聖堂 / 修道院 / 寺社 / 天文台 / 墓所 / 記念塔` のどれとも断定できない
- 内部へ進むほど、石、骨、根、膜、星図、器官の区別が曖昧になる

### 5.2 塔の役割

| 層 | 役割 |
|----|------|
| 表層 | 世界移動の基点、冒険のハブ |
| 中層 | 記録・所属・通行の審査装置 |
| 深層 | 継ぎ目を保つために何かを受け取り続ける器官 |

### 5.3 生きた門の状態遷移

| 状態 | 条件 | 見た目 | 物語機能 |
|------|------|--------|----------|
| **Dormant** | 未解放 | 石・骨・殻に見える | 謎の造形物として見せる |
| **Listening** | 条件の一部を満たす | 微振動、湿り気、眼のような孔 | 反応し始める不気味さ |
| **Awakened** | 解放条件達成 | 脈動、音、光、呼気 | 世界移動可能 |
| **Hungry** | 歪みが蓄積 | 反応が過剰、名称や記録を乱す | 失踪や残響と接続 |
| **Ruptured** | 深層崩壊 | 門と周辺地形が一体化 | postgame 級異常領域 |

### 5.4 門の解放条件カテゴリ

初回リリースの 20+ 世界で解放条件が単調化しないよう、以下のカテゴリを配分する。

| 解放条件カテゴリ | 目安世界数 | 例 |
|------------------|-----------:|----|
| ストーリー進行 | 6 | 人物救出、勢力調停、記録回収 |
| トーナメント階位 | 4 | 上位認定、通行権付与 |
| 所属の正当化 | 3 | 偽記録是正、相続確定、弔い完了 |
| 配合・変異条件 | 3 | 系統共鳴、変異種提示、血統証明 |
| 儀礼・奉納 | 3 | 葬送、奉納、巡礼の完了 |
| 例外的強制解放 | 2+ | 事故、破損、敵対勢力の介入 |

## 6. 開始村失踪事件ファイル

### 6.1 開始村事件の canonical 数

開始村における公式に語られる失踪件数と、実際に門との関連が強い件数を分けて管理する。

| 項目 | 数 |
|------|---:|
| 伝承として数えられている失踪件数 | 9 |
| 記録上「死亡」と処理された件数 | 2 |
| 記録上「転出」と処理された件数 | 3 |
| 記録上「不明」と処理された件数 | 4 |
| 主人公が生まれる前の失踪 | 6 |
| 主人公の生存中に起きた失踪 | 3 |
| 最後の確認失踪から現在までの年数 | 27年 |

### 6.2 共通項

9件のうち、以下の共通項を複数持つ者ほど門との関連が強い。

| 共通項 | 該当件数 | 備考 |
|--------|---------:|------|
| 改名、通称変更、屋号移動の履歴がある | 7 | 婚姻、養子、奉公替えを含む |
| 家系図と村の口伝で親子関係が一致しない | 6 | 継子、拾い子、死児の名継ぎを含む |
| 所属札、家畜札、土地札の付け替えに関わっていた | 5 | 家畜番主人公の視点と接続 |
| 葬送や供養が未完、または簡略化されていた家に属する | 4 | 死者と失踪者の扱いが混ざる |
| 塔の麓の古い境界祭に参加していた | 6 | 儀礼の意味は失われている |
| 闘技登録や商会登録など、外部所属が二重化していた | 3 | 世界観拡張の予告 |

### 6.3 物語上の意味

- 失踪は `塔の気まぐれ` ではない
- 村人は無知な被害者だけでもない
- 共同体は `誰が危ないか` の傾向を薄々知りながら、制度としては記録しなかった

## 8. 主人公設計

### 8.1 基本プロフィール

| 項目 | 内容 |
|------|------|
| 年齢 | 45歳 |
| 性別 | 選択式 |
| 発話 | 無言主人公 |
| 職能 | 家畜番 |
| 出自 | 小さな村の住人 |
| 冒険開始理由 | 自分の意思で塔へ向かう |
| 長年の動機 | 子どもの頃から「近づくな」と言われ続け、その理由に納得していない |
| 対塔感情 | 恐怖、執着、生活の延長としての違和感が混ざる |

### 8.2 家畜番であることの物語的利点

| 項目 | 物語効果 |
|------|----------|
| 個体識別に慣れている | 札、焼印、所属のズレに敏感 |
| 命を数える仕事 | 配合と倫理の問題に鈍感でも綺麗事でもない |
| 村の生活圏を知る | 禁忌が空気ではなく生活規範として見える |
| 低い身分すぎない | 村の中心にいないが、周辺事情に触れやすい |

### 8.3 内的アーク

| 軸 | 初期 | 終盤 |
|----|------|------|
| **認識** | 塔は気になるが触れないもの | 塔は共同体が使い続けてきた構造の表面だと知る |
| **倫理** | 命を世話し、数え、失うことに慣れている | 数えるだけでは済まない命を引き受ける |
| **社会** | 村の空気を破れない | 空気より記憶と責任を選ぶ |
| **存在** | 自分は普通の人間だと思っている | 普通の人間だからこそ終わらせる側に立つ |

### 8.4 無言主人公の演出規則

| 手段 | 使い方 |
|------|--------|
| 選択肢 | 断定よりも態度差、責任感、拒否感を表現する |
| 間 | 年齢と重さを出すため、即答しない間を活かす |
| 仕草 | 札を見る、名簿に触れる、家畜へ手を伸ばす等で内面を示す |
| 他者反応 | NPCが主人公を見て勝手に読み取ることで、無言でも社会的位置を見せる |

## 10. 5幕構成

### 10.1 全体配分

| 幕 | 目安世界数 | プレイ時間 | 主機能 |
|----|-----------:|-----------:|--------|
| Prologue | 1 | 1〜2h | 日常、塔、失踪の傷 |
| Act I | 3 | 6〜8h | 反復する禁忌の発見 |
| Act II | 5 | 12〜16h | 制度と勢力の露出 |
| Act III | 5 | 14〜18h | 分かりやすい悪と誤答 |
| Act IV | 4 | 12〜16h | 残響と配合倫理の接続 |
| Act V | 3+ | 8〜12h | 人間の方法を断つ決着 |
| Postgame | 2+ 深層領域 | 20h+ | 世界そのものの由来へ降りる |

### 10.2 Prologue: 小さな村の傷

**物語目的**

- 村の生活と塔の異物感を並置する
- 主人公が「自分から」行く理由を成立させる
- 家畜札、家名、失踪者名簿、境界祭の欠片を見せる

**解決する質問**

- なぜ村は塔を嫌うのか

**残す質問**

- 失踪者の共通項は何か
- なぜそれが門と関係するのか

### 10.3 Act I: 同じ傷を持つ世界

**物語目的**

- 開始村だけの怪談ではないと示す
- 3つの世界で「別の言葉の同じ禁忌」を提示する
- 楽しい冒険と不気味な反復を同時に体験させる

**開示する真実**

- 改名、継子、所属の揺らぎが各世界で禁忌化されている
- 門はただの通路ではなく、条件に反応している

**誤認させること**

- 世界ごとの迷信がたまたま似ているだけだと思わせる

### 10.4 Act II: 人間の制度が見え始める

**物語目的**

- 宗教、記録、行政、交易の四方向から塔への理屈を見せる
- 人間が失踪を無知で放置しているだけではないと分からせる

**開示する真実**

- 記録改竄が複数世界で起きている
- 闘技会や通行制度は一部で選別装置として機能している
- 配合は文化的娯楽でありながら制度運用とも接続している

**誤認させること**

- 記録官や神官の一派だけが悪だと見せかける

### 10.5 Act III: 分かりやすい悪、間違った答え

**物語目的**

- カルト、密輸、禁配合市場など、わかりやすい悪の顔を提示する
- プレイヤーに一度「これを潰せば終わる」と思わせる

**開示する真実**

- 現在の悪意ある利用者は存在する
- しかし彼らは既存構造の寄生者であって起点ではない

**誤認させること**

- 事件の主因を現代の犯罪組織に限定してしまうこと

### 10.6 Act IV: 失踪者は終わっていない

**物語目的**

- 失踪が `死` ではなく `残響化` である可能性を本格化する
- 一部モンスター挙動と失踪者の痕跡を繋ぐ
- 配合の倫理問題をシステムと物語で重ねる

**開示する真実**

- 継ぎ目は何かを受け取り続けている
- 失踪者は消費されたのではなく、固定の失敗として残されている

**誤認させること**

- 全員を完全に元へ戻せると思わせること

### 10.7 Act V: 終わらせるべきは仕組み

**物語目的**

- 黒幕退治ではなく、方法と運用の停止を本編の決着にする
- 村の失踪を共同体の記憶へ戻す
- 塔を神聖視も破壊もせず、喰わせ続けるやり方を断つ

**本編で完結させる答え**

- 村人失踪の意味
- 人間側の制度的責任
- 主人公が塔へ向かった理由への応答
- 主人公自身が日常へどう戻るか、あるいは戻れないか

**残す深層問い**

- 原初世界の設計者は何だったのか
- 塔は誰かの遺産なのか、世界の器官なのか
- モンスターと人間の境界は最初から分かれていたのか

## 12. 禁忌ルール

### 12.1 全世界共通の深層禁忌

| 禁忌 | 深層理由 | 日常形への変形 |
|------|----------|----------------|
| 継ぎ子を境へ立たせるな | 所属が不安定な者は門と共鳴しやすい | 子守歌、婚礼戒め、差別語 |
| 改名直後に門をくぐるな | 名の固定が済んでいない | 旅立ち前の改名待ち期間 |
| 焼けた記録を使うな | 破棄された所属が再活性化する | 古文書忌避、焼札禁制 |
| 失踪者の旧名を軽く呼ぶな | 残響を引く可能性 | 弔い詞、代名詞置換 |
| 葬送未完の者を塔へ近づけるな | 所属と終わりが固定されていない | 埋葬税、葬祭規則 |
| 無印の個体を門に通すな | 所属不明は危険 | 通行証、家畜札、印章制度 |

### 12.2 タブー運用の4階層

| 層 | 形 | 例 |
|----|----|----|
| 家庭 | しつけ、言い伝え | 「その名で呼ぶな」 |
| 村落 | 祭礼、婚姻、葬送 | 継ぎ子を祭りへ出さない |
| 行政 | 台帳、通行、認可 | 改名後の移動制限 |
| 宗教 | 穢れ、供養、聖別 | 失踪者名の封印 |

## 14. Initial Release 結末と Postgame 深化

### 14.1 Initial Release で完結させること

1. 開始村失踪事件の意味
2. 主人公が塔へ向かった理由への返答
3. 各勢力の責任の所在
4. 継ぎ目を人間がどう使い続けてきたか
5. その方法を止める決断と実行

### 14.2 Postgame に送ること

| 深化テーマ | 内容 |
|------------|------|
| 原初世界 | 大分裂以前の構造 |
| 残響救済 | どこまで回復可能か |
| 高位変異 | モンスター変異と残響の接点 |
| 塔の多重解釈 | 建築、器官、神殿、墓所の全てが成立する理由 |
| 人間 / モンスター境界 | 図鑑と血統の根本見直し |

### 14.3 エンディングトーン

- 勧善懲悪の勝利ではない
- 共同体の痛みを消さない
- それでも `見て見ぬふりを続けるよりは正しい` と感じられる終わりにする

## 15. ストーリー実装ガイド

### 15.1 事実配置の原則

- 一つの重要事実は `最低3媒体` で置く
- うち一つは必ず `生活のディテール` から見せる
- 露骨な説明イベントは 1事実あたり最大1回まで

### 15.2 伏線配置の比率

| 配置先 | 目安比率 |
|--------|---------:|
| NPC会話 | 33% |
| 記録物 | 22% |
| 建築 / 地形 | 17% |
| 図鑑 / 配合 / バトル演出 | 17% |
| イベント断章 | 11% |

### 15.3 シナリオ QA チェック

- Prologue だけで `主人公が自分から行く理由` が成立しているか
- 各幕に `新情報`, `誤認`, `修正` があるか
- 各世界に `名前 / 所属 / 継承 / 門` のどれかの変奏が必ずあるか
- 本編ラスボス戦が単なる物理的強敵でなく、構造の決着と結びついているか
- postgame の深掘りが、本編未完の穴埋めになっていないか
