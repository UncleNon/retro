## 1. 文書目的

この文書は、以下を一本化する。

1. 本作の物語が何を描く作品なのか
2. 世界がどういう法則で成り立っているのか
3. 主人公が何を知り、何を誤解し、何を選び取るのか
4. プレイヤーに何をいつ教え、何を残すのか
5. Initial Release で何を完結させ、何を postgame deep lore に送るのか

本作の物語は、`雰囲気先行のダークファンタジー` ではなく、`共同体・制度・継承・名付け・配合・世界越境` が一つの論理で繋がる構造物として設計する。

## 2. 作品の核

### 2.1 一文定義

> 小さな村で家畜番として生きてきた45歳の無言主人公が、昔から禁忌とされ、村人失踪の記憶がまとわりつく塔へ自ら踏み込み、人とモンスターと世界の「継ぎ足し」の歴史を暴き、共同体が何代にもわたり続けてきた沈黙の方法を終わらせる物語。

### 2.2 プレイヤーに最終的に残したい感情

| 項目 | 目標 |
|------|------|
| **序盤** | 懐かしいのに妙に生々しい |
| **中盤** | 世界が広がるほど同じ傷跡が見えてくる |
| **終盤** | 悪人を倒すだけでは終わらない責任がある |
| **クリア後** | 物語は完結したが、世界はまだ底が見えない |

### 2.3 中心テーマ

| テーマ | 定義 | 物語への落とし込み |
|--------|------|--------------------|
| **継承と断絶** | 名、家、技、血、所属、世界は継がれるが、継ぎ方を誤ると歪む | 配合、養子、相続、門、世界分裂が同じ構文で語られる |
| **管理された命** | 生き物を数える、札を付ける、所有する、登録する行為の暴力 | 家畜番主人公、戸籍、家畜札、闘技会登録、図鑑 |
| **共同体の沈黙** | 人々は悪意だけでなく生活防衛のために真実を隠す | 長老、記録官、宗教者、商人がそれぞれ別の理屈で沈黙を正当化する |
| **大人の冒険** | 若さの衝動ではなく、長年の違和感に責任を取る旅 | 主人公45歳、無言、大げさな英雄演出を避ける |
| **わからなさの維持** | 全部を解説しないことで世界の深さを保つ | 上位存在、原初世界、塔の意志は初回で断定しない |

### 2.4 避けるべき安易さ

1. 血筋だけで主人公を特別扱いしない
2. 黒幕一人で全問題を回収しない
3. 塔を科学装置や古代兵器に矮小化しない
4. ダークさを残酷描写の量で表現しない
5. postgame 用の謎を理由に本編を未完にしない

## 3. Canonical Truth Contract

この作品は「何が確定真実で、何を意図的に未確定に保つか」を契約として管理する。

### 3.1 Initial Release で真実として確定させるもの

| 項目 | 確定内容 |
|------|----------|
| 主人公の立場 | 塔の外の普通の生活圏に属する、45歳の家畜番 |
| 村の禁忌 | 迷信ではなく、実際の村人失踪に基づく共同体の記憶 |
| 失踪の共通項 | `名前 / 家系 / 所属` のいずれかに歪みがある者へ門が反応しやすい |
| 人間側の責任 | 複数勢力が歪みを知りつつ、記録・宗教・制度・物流で隠し続けてきた |
| 配合との接続 | 配合は単なる育成技術ではなく、世界の継ぎ足し法則を人間が利用したもの |
| 本編の結末 | 失踪を支えてきた人間社会の運用を止め、共同体の沈黙を破る |

### 3.2 Initial Release で断定しないもの

| 項目 | 理由 |
|------|------|
| 上位存在の正体 | 神、原初生命、世界自体の自己修復機構など、断定すると世界が小さくなる |
| 原初の単一世界の正確な成り立ち | postgame deep lore の主題に残す |
| 塔に人格や意思があるか | あるようにも見え、無いようにも見える余白が必要 |
| モンスターと人間の原初境界 | 図鑑・変異・残響で示すが、解答は残す |

### 3.3 Story QA 原則

- 本編だけで `事件`, `責任`, `選択`, `終止符` が成立していること
- postgame の役割は `補完` ではなく `深化`
- 主要謎は、最低 `3媒体 × 3回` で異なる角度から提示すること

### 3.4 Release Scope Contract

仕様書で `20+世界` と書くのは長期方針を示すためだが、制作計画と content budget では以下を canonical baseline とする。

| 項目 | canonical baseline |
|------|--------------------|
| **North Star** | `20+ mainline worlds + deep anomaly regions` |
| **Initial Release** | `21 mainline worlds (W-001〜W-021)` |
| **Postgame** | `2+ anomaly regions` を追加。`W-022+` 相当の deep zone として別 budget で扱う |
| **MVP** | `5 validation worlds`。`W-001〜W-005` 相当の早期帯から切り出すが、出荷 world 数そのものではない |

- 要件文書では引き続き `20+` と表現してよい
- ただし spec, world catalog, content budget, production tracking は `21 mainline worlds` を基準値として揃える
- `Postgame` の深層領域は mainline 世界数と混ぜず、`anomaly region` として別管理する

## 4. 世界法則

### 4.1 四つの根

この世界の深層では、以下の四つが一本の法則で結ばれている。

1. **Name**: 何者として呼ばれるか
2. **Lineage**: どの流れから受け継いだか
3. **Affiliation**: どこに属すると記録されているか
4. **Gate**: 境界をまたぐための器官

### 4.2 用語定義

| 用語 | 定義 |
|------|------|
| **真名** | 儀礼・契約・弔いの場で効力を持つ名 |
| **通称** | 日常で使う名 |
| **家名** | 家門、屋号、家畜印、戸籍台帳に紐づく名 |
| **系譜** | 血縁だけでなく、養子、継子、婚姻、師弟、奉仕関係まで含む継承線 |
| **所属** | 家、寺院、商会、軍、闘技局、村落など、外部に証明される帰属 |
| **残響** | 失踪者が完全消滅せず、門の向こうの継ぎ目に残る痕跡 |
| **継ぎ足し** | 異なるものを結び、ずれごと成立させる行為。配合もその一形態 |

### 4.3 深層法則

| 法則ID | 名称 | 内容 |
|--------|------|------|
| L-01 | **名の固定** | 生き物は名によって世界内の位置を固定される |
| L-02 | **継承の重さ** | 継承は権利だけでなく負荷も受け継ぐ |
| L-03 | **所属の空白** | 所属が二重、空白、偽装になった者は境界に引っかかる |
| L-04 | **門の共鳴** | 門は人格ではなく、継ぎ目の不安定さに反応する |
| L-05 | **残響の保持** | 失踪は即死ではなく、継ぎ目側に痕跡が残りうる |
| L-06 | **配合の模倣** | 配合は門や継ぎ足しの法則を人間社会へ縮小実装したもの |

### 4.4 生活レベルでの迷信化

深層法則は、庶民の生活では以下のような形に変質して流通している。

| 深層法則 | 庶民レベルの言い回し |
|----------|----------------------|
| 名の固定 | 改名した者をすぐ門へ近づけるな |
| 継承の重さ | 継ぎ子を境目へ立たせるな |
| 所属の空白 | 帰る家のない者を塔へやるな |
| 門の共鳴 | 焼けた印章を持って塔へ行くな |
| 残響の保持 | 行方知れずの者の名を軽々しく呼ぶな |
| 配合の模倣 | 血の合わぬものを混ぜるな |

### 4.5 モンスター存在論の五層

Initial Release では、モンスターの全起源を一つに断定しない。その代わり、遭遇する個体を次の五層で整理できるようにする。

| 層 | 内容 | プレイヤーの理解 | 設計ルール |
|----|------|------------------|------------|
| **wildborn** | 大分裂後の世界で安定化し、生態系に根づいた在来系 | ふつうの野生・家畜に見える | 生活圏の説得力の土台。各世界の最低6割はここから作る |
| **gate-touched** | 門や継ぎ目の長期曝露で形質がずれた系統 | 門の近くで妙に挙動が変わる | 色、鳴き、体表、出生地で異常を匂わせる |
| **bred-line** | 人間社会が配合・飼養・選抜で固定した系統 | 便利で扱いやすい育成文化の顔 | 血統証、闘技、市場価格と強く結ぶ |
| **record-bent** | 名札、印章、所属記録に反応する個体群 | なぜか札や帳面に執着する | `名前 / 所属 / 印` を世界観フレーバーから戦闘挙動まで接続する |
| **remnant-bearing** | 失踪者や場所の残響を薄く宿す個体 | 人間に似ているとは断定できない違和感 | レア扱い。全体の中核謎には使うが、量産しすぎない |

- `すべてのモンスターが元人間` という読みにはしない
- `remnant-bearing` は一部の個体や変異系列に限定し、世界全体の空気を支える程度に留める
- 同じ種でも世界によって `wildborn` と `gate-touched` の両方がありうる
- `bred-line` は人工生命ではなく、`継ぎ足し法則を人間が都合よく飼いならした結果` として扱う

### 4.6 歪みの観測徴候

世界ごとの差異を作っても、`門に近い歪み` の見え方には共通の癖を持たせる。
これにより、プレイヤーは説明を受ける前に「別世界でも同じ種類の傷だ」と気づける。

| 徴候ID | 現場の見え方 | 生活者の言い換え | 設計上の用途 |
|--------|--------------|------------------|--------------|
| SG-01 | 呼ばれた名が役名、旧名、家名へ滑る。会話の途中で言い直しが起きる | 言い間違い、昔癖、礼儀 | `名` と `所属` の揺れを会話で見せる |
| SG-02 | 台帳、札、焼印、銘板が湿ったように滲む。削り跡だけが新しく見える | ただの湿気、塩気、保管不良 | 記録改竄と門反応の接続を物で見せる |
| SG-03 | 家畜が境界線で止まる。鳥が杭に止まらず、虫の音だけが切れる | 風向きが悪い、獣が臆病 | `門` と生活圏の境界を非言語で示す |
| SG-04 | 鈴、札、布片、灰、骨片が夜のうちにしきい値側へ寄る | 子どものいたずら、掃除不足、鼠の仕業 | `所属を固定する物` が境界へ吸われる感じを出す |
| SG-05 | 姿より先に声、足音、役名が届く。方向感覚だけが一拍ずれる | 谷の反響、鐘の返り、気のせい | `残響` が死とは別の現象であることを早めに匂わせる |

- 1世界につき `徴候2種以上` を必須にする
- うち1つは `物理徴候`、1つは `会話 / 運用徴候` にする
- 終盤世界では SG-04, SG-05 を増やし、序盤世界では SG-01, SG-03 を中心に使う

## 5. 塔と生きた門

### 5.1 塔の定義

- 塔は世界外の抽象空間ではない
- 山、谷、畑、牧草地、道、村、城がある通常のファンタジー地形の中に異物として立つ
- 外観は `聖堂 / 修道院 / 寺社 / 天文台 / 墓所 / 記念塔` のどれとも断定できない
- 内部へ進むほど、石、骨、根、膜、星図、器官の区別が曖昧になる

### 5.2 塔の役割

| 層 | 役割 |
|----|------|
| 表層 | 世界移動の基点、冒険のハブ |
| 中層 | 記録・所属・通行の審査装置 |
| 深層 | 継ぎ目を保つために何かを受け取り続ける器官 |

### 5.3 生きた門の状態遷移

| 状態 | 条件 | 見た目 | 物語機能 |
|------|------|--------|----------|
| **Dormant** | 未解放 | 石・骨・殻に見える | 謎の造形物として見せる |
| **Listening** | 条件の一部を満たす | 微振動、湿り気、眼のような孔 | 反応し始める不気味さ |
| **Awakened** | 解放条件達成 | 脈動、音、光、呼気 | 世界移動可能 |
| **Hungry** | 歪みが蓄積 | 反応が過剰、名称や記録を乱す | 失踪や残響と接続 |
| **Ruptured** | 深層崩壊 | 門と周辺地形が一体化 | postgame 級異常領域 |

### 5.4 門の解放条件カテゴリ

初回リリースの 20+ 世界で解放条件が単調化しないよう、以下のカテゴリを配分する。

| 解放条件カテゴリ | 目安世界数 | 例 |
|------------------|-----------:|----|
| ストーリー進行 | 6 | 人物救出、勢力調停、記録回収 |
| トーナメント階位 | 4 | 上位認定、通行権付与 |
| 所属の正当化 | 3 | 偽記録是正、相続確定、弔い完了 |
| 配合・変異条件 | 3 | 系統共鳴、変異種提示、血統証明 |
| 儀礼・奉納 | 3 | 葬送、奉納、巡礼の完了 |
| 例外的強制解放 | 2+ | 事故、破損、敵対勢力の介入 |

### 5.5 門の状態が生む生活症状

門の状態は演出だけでなく、世界ごとの禁忌、失踪、モンスター生態へ具体的に波及する。

| 門状態 | 生活圏の症状 | モンスター側の症状 | 物語での使い方 |
|--------|--------------|--------------------|----------------|
| **Dormant** | 昔話だけ残り、禁忌の理由が風化する | 在来種に大きな異常は出ない | Prologue, echo_world の基準状態 |
| **Listening** | 名の言い換え、札の付け替え、祭礼の細則が増える | `record-bent` 個体が増え、印や呼び声に反応する | Act I〜II の違和感を支える |
| **Awakened** | 通行権、儀礼、奉納が制度化される | `gate-touched` や `bred-line` の価値が上がる | judgment_world, ritual_world の基盤 |
| **Hungry** | 失踪、転出偽装、二重所属が増え、共同体が沈黙を選ぶ | `remnant-bearing` や異常変異が観測される | fracture_world の主症状 |
| **Ruptured** | 風土、記録、地形が一体で壊れ、境界概念が崩れる | family 境界の読めない個体や深層種が出る | mainline 最終境界世界と postgame 深層領域に限定する |

- `Hungry` までは本編で主症状として扱う
- `Ruptured` は `W-021` で本編の最終境界として触れるが、完全な説明は postgame に送る
- 各世界は `gate state` を一つ主状態として持ち、門条件、禁忌、モンスターの価値付けをそこから逆算する

## 6. 開始村失踪事件ファイル

個別 case の固定は `07_starting_village_incident_and_silence_matrix.md` を canonical とし、本節は集計値と物語機能を保持する。

### 6.1 開始村事件の canonical 数

開始村における公式に語られる失踪件数と、実際に門との関連が強い件数を分けて管理する。

| 項目 | 数 |
|------|---:|
| 伝承として数えられている失踪件数 | 9 |
| 記録上「死亡」と処理された件数 | 2 |
| 記録上「転出」と処理された件数 | 3 |
| 記録上「不明」と処理された件数 | 4 |
| 主人公が生まれる前の失踪 | 6 |
| 主人公の生存中に起きた失踪 | 3 |
| 最後の確認失踪から現在までの年数 | 27年 |

### 6.2 共通項

9件のうち、以下の共通項を複数持つ者ほど門との関連が強い。

| 共通項 | 該当件数 | 備考 |
|--------|---------:|------|
| 改名、通称変更、屋号移動の履歴がある | 7 | 婚姻、養子、奉公替えを含む |
| 家系図と村の口伝で親子関係が一致しない | 6 | 継子、拾い子、死児の名継ぎを含む |
| 所属札、家畜札、土地札の付け替えに関わっていた | 5 | 家畜番主人公の視点と接続 |
| 葬送や供養が未完、または簡略化されていた家に属する | 4 | 死者と失踪者の扱いが混ざる |
| 塔の麓の古い境界祭に参加していた | 6 | 儀礼の意味は失われている |
| 闘技登録や商会登録など、外部所属が二重化していた | 3 | 世界観拡張の予告 |

### 6.3 物語上の意味

- 失踪は `塔の気まぐれ` ではない
- 村人は無知な被害者だけでもない
- 共同体は `誰が危ないか` の傾向を薄々知りながら、制度としては記録しなかった

## 7. 歴史年表

### 7.1 大年代区分

| Era | 年代 | 名称 | 世界状態 |
|-----|------|------|----------|
| E0 | 3000年前以前 | 名無き時代 | 世界の境界がまだ柔らかい |
| E1 | 約3000〜2400年前 | 原初統合期 | 世界は大きくひとつ、門の原型が生まれる |
| E2 | 約2400〜1800年前 | 境界祭儀期 | 名、所属、儀礼が門と結びつく |
| E3 | 約1800〜1000年前 | 管理共同体期 | 継ぎ目を管理する組織が生まれる |
| E4 | 約1000年前 | **大分裂** | 世界が多数化し、塔と門が傷跡として残る |
| E5 | 約1000〜520年前 | 各世界分岐期 | 政治、宗教、禁忌が世界ごとに分岐する |
| E6 | 約520〜220年前 | 制度固定期 | 記録、相続、所属制度が硬化し、歪みが見えにくくなる |
| E7 | 約220〜90年前 | 配合再命名期 | 継ぎ足し法則が「配合技術」として再流通 |
| E8 | 約90年前〜現在 | 競技化期 | 闘技会、通行権、門の公的運用が成立する |

### 7.2 歴史上の重要事件

| ID | 年代 | 事件 | 物語機能 |
|----|------|------|----------|
| H-01 | E1 | 原初の継ぎ目が初めて塞がれる | 門の起源に関わる神話層 |
| H-02 | E2 | 真名奉納儀式の体系化 | 発話禁忌の起源 |
| H-03 | E3 | 継ぎ目管理共同体の成立 | 後の塔管理組織の原型 |
| H-04 | E4 | 大分裂 | 20+ 世界の出自 |
| H-05 | E5 | 名簿改竄と継子選別が制度化 | 共同体の罪の原型 |
| H-06 | E6 | 記録官制度と寺院台帳の二重化 | 記録の食い違いを生む |
| H-07 | E7 | 配合の再命名と一般流通 | 育成文化の光と闇 |
| H-08 | E8 | 闘技会の公制度化 | 選別が娯楽に擬装される |

### 7.3 歴史上の三つの罪

| 罪 | 内容 | 現代での変形 |
|----|------|--------------|
| **選別** | 継ぎ目に触れやすい者を共同体が切り離した | 迷信、差別、養子忌避 |
| **改竄** | 失踪を死・転出・不在に書き換えた | 台帳不一致、相続歪み |
| **再包装** | 危険な継ぎ足し法則を無害な育成技術として再流通させた | 配合文化、闘技会、血統主義 |

### 7.4 大分裂以後 1000 年の運用原則

本編でプレイヤーが触る `苦さ`, `利害`, `歴史の重み` は、原初神話ではなく `大分裂以後の 1000 年` の積み重ねで説明する。
この 1000 年で固定する政治史の骨組みは、以下の 6 層で扱う。

1. **避難民の仮登録**: 借り名、仮親、仮所属で人を数えた結果、後の差別と失踪選別の母型ができた
2. **弔いと記録の独占**: 死と失踪を言い換える実務が、宗教と台帳を太らせた
3. **海路と通行税の制度化**: 越境は危険管理であると同時に、国家と商人の収益源になった
4. **相続戦と二重登録**: 家を守るための婚姻、養子、記録分立が、`名 / 家系 / 所属` のずれを固定した
5. **配合の再命名**: 継ぎ足し法則を「便利な技術」として再包装し、生活圏へ埋め込んだ
6. **門競と公的選別**: 事故防止を名目に、適応者観測とふるい落としを娯楽化した

このレイヤを国、勢力、人物、現行関係まで含めて具体化した canonical 文書を
`docs/specs/story/06_millennial_geopolitics_and_personages.md` とする。

### 9.3 反復モチーフの grammar

20+ 世界を跨いでも同じ作品に見せるため、反復モチーフを先に固定する。
同じ意味を同じ物で言い続けるのではなく、`材質 / 用途 / 所有者` をずらして反復させる。

| motif | 表層での見え方 | 深層での意味 | 主な出し先 |
|-------|----------------|--------------|------------|
| **削り跡** | 削れた名札、消えた表札、磨耗した刻印 | 事実を消した痕跡は完全には消えない | 村、記録庫、宿帳、家畜札 |
| **二重印** | 重ね押し印章、二つの家印、裏表の布札 | 所属の二重化と責任転嫁 | 官僚世界、商業世界、闘技登録 |
| **鈴と沈黙** | 家畜鈴、巡礼鈴、閉じた鐘楼、無音域 | 何が数えられ、何が数えられなくなったか | 牧畜世界、巡礼世界、塔都 |
| **湿りと塩** | 湿った台帳、塩庫、曇る鏡、水脈 | 記録や名が固定しきらずに滲む状態 | 港湾、修院、鏡森、塩庫 |
| **布と覆い** | 黒布、家印布、顔覆い、旗 | 名や顔を隠しつつ所属だけ残す操作 | 巡礼、墓苑、名家、闘技会 |
| **継ぎ目** | 焼継ぎ器、骨の籠、縫い直し、割れた門 | 世界と命が継ぎ足しで保たれていること | 工芸世界、深層門、モンスター図鑑 |

## 10. 5幕構成

### 10.1 全体配分

| 幕 | 目安世界数 | プレイ時間 | 主機能 |
|----|-----------:|-----------:|--------|
| Prologue | 0 mainline worlds + 村 / 塔 | 1〜2h | 日常、塔、失踪の傷 |
| Act I | 4 | 6〜8h | 反復する禁忌の発見 |
| Act II | 5 | 12〜16h | 制度と勢力の露出 |
| Act III | 5 | 14〜18h | 分かりやすい悪と誤答 |
| Act IV | 5 | 12〜16h | 残響と配合倫理の接続 |
| Act V | 2 | 8〜12h | 人間の方法を断つ決着 |
| Postgame | 2+ 深層領域 | 20h+ | 世界そのものの由来へ降りる |

### 10.2 Prologue: 小さな村の傷

**物語目的**

- 村の生活と塔の異物感を並置する
- 主人公が「自分から」行く理由を成立させる
- 家畜札、家名、失踪者名簿、境界祭の欠片を見せる

**解決する質問**

- なぜ村は塔を嫌うのか

**残す質問**

- 失踪者の共通項は何か
- なぜそれが門と関係するのか

### 10.3 Act I: 同じ傷を持つ世界

**物語目的**

- 開始村だけの怪談ではないと示す
- 3つの世界で「別の言葉の同じ禁忌」を提示する
- 楽しい冒険と不気味な反復を同時に体験させる

**開示する真実**

- 改名、継子、所属の揺らぎが各世界で禁忌化されている
- 門はただの通路ではなく、条件に反応している

**誤認させること**

- 世界ごとの迷信がたまたま似ているだけだと思わせる

### 10.4 Act II: 人間の制度が見え始める

**物語目的**

- 宗教、記録、行政、交易の四方向から塔への理屈を見せる
- 人間が失踪を無知で放置しているだけではないと分からせる

**開示する真実**

- 記録改竄が複数世界で起きている
- 闘技会や通行制度は一部で選別装置として機能している
- 配合は文化的娯楽でありながら制度運用とも接続している

**誤認させること**

- 記録官や神官の一派だけが悪だと見せかける

### 10.5 Act III: 分かりやすい悪、間違った答え

**物語目的**

- カルト、密輸、禁配合市場など、わかりやすい悪の顔を提示する
- プレイヤーに一度「これを潰せば終わる」と思わせる

**開示する真実**

- 現在の悪意ある利用者は存在する
- しかし彼らは既存構造の寄生者であって起点ではない

**誤認させること**

- 事件の主因を現代の犯罪組織に限定してしまうこと

### 10.6 Act IV: 失踪者は終わっていない

**物語目的**

- 失踪が `死` ではなく `残響化` である可能性を本格化する
- 一部モンスター挙動と失踪者の痕跡を繋ぐ
- 配合の倫理問題をシステムと物語で重ねる

**開示する真実**

- 継ぎ目は何かを受け取り続けている
- 失踪者は消費されたのではなく、固定の失敗として残されている

**誤認させること**

- 全員を完全に元へ戻せると思わせること

### 10.7 Act V: 終わらせるべきは仕組み

**物語目的**

- 黒幕退治ではなく、方法と運用の停止を本編の決着にする
- 村の失踪を共同体の記憶へ戻す
- 塔を神聖視も破壊もせず、喰わせ続けるやり方を断つ

**本編で完結させる答え**

- 村人失踪の意味
- 人間側の制度的責任
- 主人公が塔へ向かった理由への応答
- 主人公自身が日常へどう戻るか、あるいは戻れないか

**残す深層問い**

- 原初世界の設計者は何だったのか
- 塔は誰かの遺産なのか、世界の器官なのか
- モンスターと人間の境界は最初から分かれていたのか

## 12. 禁忌ルール

### 12.1 全世界共通の深層禁忌

| 禁忌 | 深層理由 | 日常形への変形 |

| 無印の個体を門に通すな | 所属不明は危険 | 通行証、家畜札、印章制度 |

| Prologue | 村に実害があった、塔は異常、長年の禁忌がある | 失踪の法則、他世界との共通性 |
| Act I 終了 | 村だけの話ではない、同じ禁忌が散在する | 人為と制度の関与 |
| Act II 終了 | 記録・宗教・行政が隠している | 失踪後の状態 |
| Act III 終了 | 現代の悪用者だけが原因ではない | 世界法則の中心 |
| Act IV 終了 | 失踪者は残響化しうる、配合と門は繋がる | 原初起源 |
| Clear | 人間側の責任と停止すべき運用 | 世界の始原的真相 |
| Postgame | 原初に近い層へ接続 | 最後の断定は残す |

### 13.3 沈黙の経済

真相が長く隠れ続けた理由は、`恐怖` だけではなく `運用上の利益` があったからである。
どの勢力も世界を壊そうとしているのではなく、`今さら止めるともっと壊れる` という理屈で自分を正当化している。

| 担い手 | 守っている表の価値 | 真相露出で失うもの | 自分への言い訳 | プレイヤーへの見え方 |
|--------|--------------------|--------------------|----------------|----------------------|
| 村落長老ネットワーク | 婚姻、祭礼、相互扶助、村のまとまり | 家ごとの傷、選別の記憶、共同体の信用 | 「全部言えば村が先に壊れる」 | 優しいが話を切る、数を言い直す |
| 塔保全機構 | 通行安全、交易維持、事故最小化 | 門運用の正統性、保全記録、国家的信頼 | 「誰かを選ばないと全員が危ない」 | 冷徹で実務的、悪人に見えやすい |
| 記録官連盟 | 相続、徴税、通行証、身分秩序 | 土地権利、戸籍整合、過去の改竄責任 | 「事実より整合のほうが人を守る」 | 書類は揃うが人間が消える |
| 帰還なき祈祷会群 | 弔い、慰め、信仰、巡礼経済 | 教義の正当性、奉納、慰めの言葉そのもの | 「神意にしないと遺族が耐えられない」 | 優しい語彙で加害を覆う |
| 闘技会運営局 | 娯楽、名誉、通行資格、若者の上昇路 | 人気、収益、観測データ、国家的威信 | 「競技化したほうがまだ公正だ」 | 明るい表層のまま選別を回す |
| 門越え密輸網 | 希少品流通、抜け道、裏情報 | 歪みの希少価値、仲介利幅 | 「どうせ止まらないなら売るほうが早い」 | 真相に近いが責任感が最も薄い |
| 家門連鎖同盟 | 血統資産、縁組、後継管理、財産集中 | 名家神話、婚姻市場、囲い込んだ家系価値 | 「良い継承先を選ぶのは秩序だ」 | 上品だが人を系譜としてしか見ない |

- 各勢力は `一部の被害を減らした実績` も持つ
- そのため、単純な告発だけでは崩れず、主人公は `代替の運用停止手段` を示す必要がある
- 誰も全体の真相を知らず、`部分知 + 部分利益` の重なりで沈黙が維持されてきた

## 14. Initial Release 結末と Postgame 深化

### 14.1 Initial Release で完結させること

1. 開始村失踪事件の意味
2. 主人公が塔へ向かった理由への返答
3. 各勢力の責任の所在
4. 継ぎ目を人間がどう使い続けてきたか
5. その方法を止める決断と実行

### 14.2 Postgame に送ること

| 深化テーマ | 内容 |
|------------|------|
| 原初世界 | 大分裂以前の構造 |
| 残響救済 | どこまで回復可能か |
| 高位変異 | モンスター変異と残響の接点 |
| 塔の多重解釈 | 建築、器官、神殿、墓所の全てが成立する理由 |
| 人間 / モンスター境界 | 図鑑と血統の根本見直し |

### 14.3 エンディングトーン

- 勧善懲悪の勝利ではない
- 共同体の痛みを消さない
- それでも `見て見ぬふりを続けるよりは正しい` と感じられる終わりにする

## 15. ストーリー実装ガイド

### 15.1 事実配置の原則

- 一つの重要事実は `最低3媒体` で置く
- うち一つは必ず `生活のディテール` から見せる
- 露骨な説明イベントは 1事実あたり最大1回まで

### 15.2 伏線配置の比率

| 配置先 | 目安比率 |
|--------|---------:|
| NPC会話 | 33% |
| 記録物 | 22% |
| 建築 / 地形 | 17% |
| 図鑑 / 配合 / バトル演出 | 17% |
| イベント断章 | 11% |

### 15.3 シナリオ QA チェック

- Prologue だけで `主人公が自分から行く理由` が成立しているか
- 各幕に `新情報`, `誤認`, `修正` があるか
- 各世界に `名前 / 所属 / 継承 / 門` のどれかの変奏が必ずあるか
- 本編ラスボス戦が単なる物理的強敵でなく、構造の決着と結びついているか
- postgame の深掘りが、本編未完の穴埋めになっていないか
