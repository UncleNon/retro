## 2. Canonical Contract

- 本書では既存文書に合わせ、地名は引き続き **開始村** と表記する
- 件数、記録分類、共通項の総数は [01_story_bible.md](./01_story_bible.md) の 6.1 / 6.2 を上書きしない
- `death` は「死亡として処理された」を意味し、遺体確認を保証しない
- `transfer` は「転出・奉公替え・婚姻移籍として記録上片付けた」を意味する
- `unknown` は「不明として残した」ではなく、`共同体が言い切る責任を避けた` 区分として扱う

### 2.1 共通項コード

| コード | 意味 |
|--------|------|
| `A` | 改名、通称変更、屋号移動の履歴がある |
| `B` | 家系図と口伝で親子関係が一致しない |
| `C` | 所属札、家畜札、土地札の付け替えに関わる |
| `D` | 葬送や供養が未完、または簡略化されている |
| `E` | 塔麓の古い境界祭に参加している |
| `F` | 村外所属が二重化している |

## 4. 村内の沈黙回路

### 4.1 レイヤ別

| レイヤ | 主体 | 実際にしていること | 村での言い訳 | 長期的な歪み |
|---------|------|--------------------|--------------|--------------|
| 口伝 | 長老層、古老の家 | 人名を避け、家単位でしか語らない | 「昔の話を掘るな」 | 失踪者が個人でなく家の汚れとして残る |
| 記録 | 記録番、巡回書記、記録官連盟末端 | `death / transfer / unknown` に押し込み、札の説明責任を切る | 「帳面を合わせるため」 | 事実より整合が優先される |
| 葬送 | 墓守、巡礼者、祈祷会の外縁 | 墓を空けたままにし、名だけを封じる | 「戻るかもしれないから」 | 死者と失踪者の境目が固定されない |
| 生活 | 畜舎主、糸売り、荷運び、嫁入り先 | 器や布や札を捨てず、しかし名は消す | 「使えるものは残す」 | 家と所有物だけが先に残る |
| 境界管理 | 柵直し、猟師、塔前見張りの名残 | 杭、柵、歌、祭りの順番だけ守る | 「向こうへ行かせないため」 | 理由を失った抑止だけが残る |
| 外部制度 | 塔保全機構、記録官連盟上層、祈祷会群 | case ごとの断片を受け取るが、村の傷として束ねない | 「広域秩序のため」 | 誰も全件責任を負わない |

### 4.2 責任の分散原則

- 開始村の誰か一人が真相を知っている構造にはしない
- `SV-03`, `SV-05`, `SV-09` は外部制度との接点が濃く、村外の責任導線として使う
- `SV-01`, `SV-04`, `SV-06`, `SV-07` は共同体内部の口伝と差別の継続を見せる
- `SV-08`, `SV-09` は主人公の記憶に近いため、序盤で説明しすぎず断片の違和感に留める

## 5. 主人公の記憶アンカー

### 5.1 8歳: `SV-07`

- 主人公が最初に「人は死ぬのでなく、数え直されることがある」と感じた case
- 大人たちは `死んだ` とは言うが、墓守は空碑を空けたままにしている
- 北道の杭が新しく打ち直された記憶が、塔への恐れを `景色の変化` として残す

### 5.2 14歳: `SV-08`

- 記録小屋の紙が一枚抜かれているのを、意味は分からないまま見ている
- 借り名で呼ばれていた娘と、村で呼ばれる名が違ったことを曖昧に覚えている
- 村外所属があると、人は `出ていったことにされる` と学ぶ

### 5.3 18歳: `SV-09`

- 主人公は家畜番として、消えた牛番の頭数と札の不一致を直接見る
- しかもその case は婚姻と家の継承に触れていたため、村全体が露骨に口をつぐむ
- この最後の case 以後 27 年、村は「触れないこと」で均衡を保ち、その静けさが現在の主人公を押し出す

### 5.4 45歳の現在

- 主人公は `最近の若者` ではなく、村が沈黙を日常へ埋め戻す全過程を見てきた世代に属する
- したがって旅立ちは `無知な好奇心` ではなく、`長年棚上げされた違和感に責任を取る行為` になる

## 7. 矛盾防止メモ

- `NPC-VIL-008` は 30 代のため、`SV-09` の未亡人ではなく **後年その家へ嫁いだ管理者** とする
- `death` 扱いの `SV-03`, `SV-07` は、遺体確認でなく共同体都合の死亡処理なので、空碑と両立する
- 表札跡と空碑はそれぞれ `単一 case の痕跡` ではなく、村が複数件を重ねて一つへ押し込んだ痕として使う
- 主人公は 18 歳時点で `SV-09` の記録不一致を見ているが、法則の全体像は知らない。知りすぎると Act I の発見が弱くなる
- 村内の誰かを単独黒幕にしない。責任は `共同体 / 記録 / 葬送 / 外部制度` の分業として描く
