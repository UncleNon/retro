## 1. 文化設計の目的

この文書の目的は、20+ 世界を「別々の見た目」ではなく、`制度と禁忌が異なる社会` として設計することにある。

すべての世界は以下の共通根を持つ。

1. `名前`
2. `家系 / 継承`
3. `所属`
4. `塔 / 門`
5. `モンスターの社会的位置`

ただし、その表れ方は世界ごとに変える。プレイヤーが体験するのは「世界ごとに違う迷信」だが、設計上の深層では同じ根が通っている必要がある。

## 2. 20+ 世界の配分指針

### 2.1 世界カテゴリ配分

Initial Release の 20+ 世界は、最低でも以下のように散らす。

| 世界カテゴリ | 最低数 | 主な役割 |
|--------------|-------:|----------|
| **牧畜 / 農耕世界** | 4 | 主人公の出自と対比しやすい生活圏 |
| **交易 / 都市世界** | 4 | 記録、所属、印章、商業の歪みを描く |
| **信仰 / 巡礼世界** | 4 | 塔禁忌の宗教化を描く |
| **軍事 / 辺境世界** | 3 | 通行管理と門の国家利用を描く |
| **異質 / サイケ世界** | 4 | 物理法則より同一性が揺らぐ領域を描く |
| **深層 / 隠し世界** | 2+ | postgame deep lore 向け |

### 2.2 世界ごとの最低差分

連続して解放される 2 世界が、以下の 5 項目で同一にならないようにする。

| 項目 | 差分ルール |
|------|------------|
| 政治 | 同じ統治方式を連続させない |
| 宗教 | 塔の意味づけを連続させない |
| 経済 | 主産業を連続させない |
| 禁忌 | 何を「してはいけないか」を変える |
| モンスター観 | 家族、資産、兵器、神使、災厄など役割を変える |

### 2.3 分類軸の重ね方

世界分類は 1 本ではなく、少なくとも次の 3 軸を同時に持つ。
これにより、`自然世界だから似る`、`story 上の役目が同じだから被る` といった混線を避ける。

| 軸 | 役割 | 例 |
|----|------|----|
| `biome / production axis` | 生活素材と景色を決める | 牧畜、港湾、巡礼、軍政、鏡森 |
| `story-function axis` | その世界が何を証明するかを決める | `echo`, `social`, `ritual`, `fracture`, `terminal` |
| `corridor axis` | 他世界とどの記号や制度で響くかを決める | 札の連鎖、弔いの連鎖、物流の連鎖 |

1 世界ごとに、必ず `3軸を1つずつ` 割り当てる。
同じ `biome` でも `story-function` と `corridor` を変えれば、別の役割を持てる。

### 2.4 反復記号バンク

20+ 世界を一本の作品として見せるため、物質記号は世界をまたいで再登場させる。
ただし、同じ意味でなく `別文脈に変形された再登場` にする。

| 記号 | 生活上の顔 | 深層で示すこと | よく効く世界 |
|------|------------|----------------|--------------|
| 札 / 印 / 焼印 | 所有、所属、通行、家畜管理 | `所属の固定` が暴力でもあること | 牧畜、都市、港湾、軍政 |
| 鈴 / 結縄 / 紐 | 数える、呼び分ける、見失わない | `名の代用品` が制度化していること | 牧畜、巡礼、葬送 |
| 灰 / 塩 / 煤 | 保存、防腐、清め、保管 | 記録や遺体を `物` として扱う癖 | 墓苑、塩庫、鍛土、港湾 |
| 布 / 面 / 覆い | 家印、巡礼装束、弔い布 | 顔や名を隠しても `所属` だけ残ること | 巡礼、神権、村落 |
| 鏡 / 水面 / 白泥 | 身だしなみ、反射、湿地の目印 | `本人と記録像のずれ` | 雪原、鏡森、深層 |
| 骨杭 / 骨樋 / 白柱 | 柵、供物台、建材、排水 | 生き物と建築の境界が曖昧なこと | 門前、港、深層、終盤世界 |

## 3. 世界を作る6軸

各世界は最低でも次の 6 軸で定義する。

| 軸 | 必須項目 |
|----|----------|
| **Power** | 誰が支配し、何で正当化するか |
| **Faith** | 塔 / 門 / 失踪をどう意味づけるか |
| **Work** | 何を作り、何で食い、何を運ぶか |
| **Blood** | 家系、養子、婚姻、継承をどう扱うか |
| **Monster** | モンスターを何として扱うか |
| **Boundary** | 塔・門・失踪者・改名者に何を禁じるか |

### 3.1 1世界あたりの最低文化要件

| 項目 | 最低数 |
|------|-------:|
| 世界固有の塔呼称 | 1 |
| 世界固有の禁忌 | 3 |
| 政治的緊張点 | 2 |
| 宗教的矛盾 | 2 |
| 経済的搾取構造 | 1 |
| モンスター観の特徴 | 1 |
| `名 / 家系 / 所属` の歪み事例 | 2 |
| 塔 / 門を巡る噂 | 4 |
| 主要勢力 | 2 |
| 深層への伏線配置点 | 3 |

## 4. 政治テンプレート

### 4.1 統治形態

| 体制 | 向く世界 | 利点 | 闇 |
|------|----------|------|----|
| **長老合議制** | 村落、山間、牧畜 | 生活感が強い | 記憶の私物化、口裏合わせ |
| **王権 + 家門制** | 王国、古都 | 家系と塔を結びやすい | 継子差別、相続改竄 |
| **官僚都市制** | 港湾、文書都市 | 記録ドラマに強い | 書類上の抹消、身分売買 |
| **神権制** | 巡礼、聖域 | 禁忌の神学化 | 失踪の神聖化、供物化 |
| **商会連合制** | 貿易、物流 | 通行証と所属に強い | 印章と所属の売買 |
| **軍政辺境制** | 火山、国境 | 門を国家資産にできる | 強制動員、隔離政策 |
| **二重権力制** | 古都 + 聖域複合 | 摩擦が生まれやすい | 責任のなすりつけ |

### 4.2 政治と塔の距離

| 距離 | 意味 | 物語効果 |
|------|------|----------|
| **公営** | 国家や自治体が塔運用を管理 | 冷たい官僚性 |
| **聖別** | 宗教側が塔を管理 | 神秘と偽装が強い |
| **黙認** | 表向き無関係だが裏で利用 | 共同責任が描きやすい |
| **放置** | 近寄らないことで均衡維持 | 村落型の怖さ |
| **争奪** | 複数勢力が塔解釈を奪い合う | 中盤以降の政治劇向き |

## 5. 宗教テンプレート

### 5.1 塔解釈のパターン

| 解釈 | 呼称例 | 主張 | 実際の機能 |
|------|--------|------|------------|
| **聖域** | 祈りの塔、封じの殿 | 近づくな、敬え | 禁忌を保存する |
| **観測所** | 星見塔、兆し台 | 空や兆候を読む場所 | 反応を観測するだけ |
| **墓所** | 還り塔、無名塚 | 帰らぬ者を納める | 失踪を弔いへ変換 |
| **継ぎ殿** | 継殿、境殿 | 境界を保つ施設 | 所属と名の審査装置 |
| **刑罰装置** | 罰の塔、赦しの塔 | 罪人を裁く場所 | 権力による排除の口実 |
| **昇天門** | 召し門、帰天楼 | 選ばれた者は上がる | 失踪の神聖化 |

### 5.2 宗教儀礼テンプレート

| 儀礼種別 | 例 | 深層接続 |
|----------|----|----------|
| 改名儀礼 | 成人、婚礼、入信 | 名の固定 |
| 継承儀礼 | 養子、家督、屋号継ぎ | 所属の移動 |
| 葬送儀礼 | 埋葬、流し、焼納 | 終了の固定 |
| 通行儀礼 | 門越え、巡礼、通行許可 | 塔の共鳴条件 |
| 奉納儀礼 | 供物、印章、獣骨、乳 | 生活物資と門の接続 |

### 5.3 宗教の腐敗ベクトル

| 腐敗パターン | 内容 |
|--------------|------|
| **意味の上書き** | 失踪を「選ばれた」と呼ぶ |
| **責任転嫁** | 共同体の加害を神意に変える |
| **排除の神学化** | 継子や改名者を穢れ扱いする |
| **記録封印** | 神殿文書だけで真実を囲い込む |

## 6. 経済テンプレート

### 6.1 基幹経済類型

| 類型 | 主要資源 | 塔 / 門との接続 |
|------|----------|-----------------|
| **牧畜経済** | 家畜、乳、皮、餌、札 | 個体識別、所有、焼印 |
| **農耕経済** | 穀物、水利、種、供物 | 境界祭、収穫祭、名寄せ |
| **港湾経済** | 通行証、商印、倉庫 | 所属と検問 |
| **鉱工経済** | 石、金属、鏡、封印具 | 印章、鍵、銘板 |
| **宗教経済** | 巡礼税、奉納、聖遺物 | 塔の正当化 |
| **配合経済** | 卵、素材、変異種、血統証 | 継承の商業化 |

### 6.2 経済と闇の接続

| 経済行為 | 闇 |
|----------|----|
| 家畜売買 | 札の付け替え、所有偽装 |
| 養子縁組 | 家名と相続の再編成 |
| 葬送商売 | 死亡 / 失踪の記録操作 |
| 配合市場 | 禁配合素材と高位変異の闇流通 |
| 通行商売 | 所属証明、印章、身分の売買 |

### 6.3 主人公視点で効く経済ディテール

家畜番主人公の視点と直結するディテールを、世界ごとに最低 2 つは入れる。

| ディテール例 | 物語機能 |
|--------------|----------|
| 家畜札の材質が世界ごとに違う | 所属の扱いの違いが見える |
| 焼印位置や耳標の意味が違う | 個体識別文化を差別化できる |
| 乳、毛、糞、骨の価値が違う | 命の扱い方を描ける |
| モンスターを家畜とみなす範囲が違う | 人間観 / モンスター観の差になる |

## 7. 家系・所属・歪みのテンプレート

### 7.1 家系の扱い方

| パターン | 特徴 | 歪みが生まれやすい点 |
|----------|------|----------------------|
| **父系家門制** | 家名が父系で継がれる | 養子、庶子、後継偽装 |
| **母系土地制** | 土地と氏が母系で継がれる | 婚姻時の所属二重化 |
| **寺院登録制** | 生誕と葬送を寺が管理 | 改宗者と未登録者の空白 |
| **商会所属制** | 家より商会が身分を決める | 退会者、債務者、偽造印 |
| **軍籍制** | 軍籍が所属の核 | 脱走者、退役者、戦災孤児 |

### 7.2 歪み類型

| 歪みID | 名称 | 内容 | 門との相性 |
|--------|------|------|------------|
| D-01 | 二重所属 | 家と寺、家と商会などに二重登録 | 高 |
| D-02 | 名の再利用 | 死者や失踪者の名を継ぐ | 高 |
| D-03 | 継子空白 | 家にいるが正式相続線に入っていない | 高 |
| D-04 | 焼失記録 | 戦災、火災、意図的破棄で所属証明が欠ける | 中 |
| D-05 | 奉公替え断絶 | 長期奉公で家系登録が切れる | 中 |
| D-06 | 儀礼未完 | 改名、婚礼、葬送、成人儀礼が完了していない | 高 |

### 7.3 物語利用原則

- 各世界で最低 1 つは `D-01〜D-06` の変奏を使う
- 開始村と同じ歪みを別文化で反復し、プレイヤーに遅れて気づかせる
- ただし同じ説明文を繰り返さず、宗教、法律、商売、家畜管理など別文脈へ変換する

## 8. モンスターの社会的扱い

### 8.1 社会類型

| 類型 | 日常での扱い | 闇 |
|------|--------------|----|
| **家族型** | 家の一員、相棒 | 喪失の隠蔽が重い |
| **家畜型** | 生産資産、労働単位 | 数合わせ、札の付け替え |
| **兵器型** | 戦力、警備、国益 | 配合の軍事利用 |
| **祭具型** | 神使、供物、媒介 | 生贄化、所有不可視化 |
| **厄災型** | 触れてはならないもの | 隔離と見せしめ |
| **証人型** | 過去を宿す存在 | 図鑑や会話で不気味さを担う |

### 8.2 法制度上の扱い

| 法的位置付け | 説明 | 向く世界 |
|--------------|------|----------|
| 財産 | 所有、売買、徴税対象 | 商業、農牧 |
| 兵役資産 | 国家徴発対象 | 軍政 |
| 半人格 | 殺傷、譲渡に制限 | 高度共存型 |
| 神聖物 | 私有禁止、教団管理 | 神権 |
| 汚染源 | 捕獲・飼育禁止 | 厄災型世界 |

### 8.3 主人公との緊張

家畜番主人公は、モンスターを `きれいごとの仲間` とも `完全な道具` とも見切れない位置にいる。この半端さを物語上の強みにする。

## 9. 社会階層

### 9.1 共通階層モデル

| 階層 | 主な構成 | 塔への距離 | 失踪リスク |
|------|----------|------------|------------|
| **統治層** | 王侯、神官長、商会長、塔管理者 | 近いが守られる | 低 |
| **実務上層** | 記録官、司祭、監督官、名家当主 | 近い | 中 |
| **生活中層** | 職人、家畜番、旅商人、闘技会運営 | 触れうる | 中 |
| **労働層** | 農夫、鉱夫、水運、奉公人 | 命令で近づけられる | 中〜高 |
| **周縁層** | 養子、流れ者、改名者、無印者、失籍者 | 保護されない | 高 |

### 9.2 重要設計ルール

1. 失踪リスクは `特別な英雄` より `制度の外縁` に集中させる
2. 支配層は真相に近いが、必ずしも全部知っているわけではない
3. 周縁層は弱者であると同時に、社会にとって都合よく使われる

## 13. 人間臭さと歴史残滓の必須項目

`制度がある` だけでは世界は生きない。
各世界は、`誰が得したか`, `誰が黙らせたか`, `何が家の中に残ったか` を持って初めて立つ。

### 13.1 世界ごとの最低要件

| 要件 | 最低数 | 例 |
|------|-------:|----|
| 失踪の受益者 | 1 | 土地相続、婚姻有利、商権、闘技枠、借財帳消し |
| 口止めの相互扶助網 | 1 | 長老、墓守、書記、荷運び、査定人が互いの弱みを握る |
| 儀礼の省略 / 代行市場 | 1 | 偽弔い人、代理婚礼、安い通行祝詞、使い回しの灰 |
| 不在者を処理する生活慣習 | 2 | 再婚待機年数、空き部屋封鎖、器の片付け時期 |
| 削除痕 / 継ぎ痕の生活物 | 2 | 削られた表札、裏返しの墓板、継ぎ当て布、平らに潰した耳標 |
| 世代差の本音 | 3層 | 老人は実被害、中年は運用、若者は競技やブランドとして消費 |

### 13.2 呼称の政治

多重名を設定するだけでは足りない。
各世界で、最低でも次を決める。

1. 誰が `生家名` で呼んでよいか
2. `幼名`, `儀礼名`, `闘技名` を使うとどういう感情が発生するか
3. 怒り、未練、支配、軽蔑を名の選択だけでどう表すか

### 13.3 家と制度のねじれ

各世界は以下のうち 2 つ以上を必ず持つ。

- 家の中の三角関係: 正嫡、継子、養親、寡婦、番頭
- 制度横断の癒着: 書記 + 神官 + 運送屋、長老 + 査定人 + 墓守
- 秘密の結びつき: 情夫 / 情婦、借財、恩義、共犯の口止め
- 生活上の差別: 席順、器の置き場、名札の材質、家畜印の位置

### 13.4 歴史残滓の置き方

各世界の町や村は、`3つの時代層` を同時に見せる。

| 層 | 何を置くか |
|----|------------|
| 大分裂直後の層 | 借り名、無銘墓、古い境界杭 |
| 中世の制度固定層 | 二重印、婚姻棚、役所と寺院の食い違い |
| 近世以降の再包装層 | 血統証、闘籍、配合見本、観戦看板 |

### 13.5 ローカル英雄とローカル厄介者

各世界は最低 1 人ずつ持つ。

- 表向きに称えられる人物
- 家の中では呪いの種として語られる人物

同一人物でもよい。
この二面性が、1000年史を教科書でなく噂話へ落とす。

## 15. 実装メモ

- 世界名、人物名、勢力名は後で変えられるが、`社会構造の役割` は先に固定する
- 一つの世界は「景色」「文化」「禁忌」「モンスター観」の 4 点で覚えられるようにする
- どの勢力も善悪二元で描かず、`守るために歪んだ理屈` を持たせる
- 世界詳細を書くときは、必ず `開始村との反復ポイント` を 1 つ以上入れる
