## 1. 目的

- モンスターを `戦う相手` で終わらせず、共同体の制度、タブー、物流、弔いの痕が **体に出た生き物** として設計する
- `生息地`, `食性`, `食物連鎖`, `人間による誤用`, `勧誘したくなる魅力` を同時に持たせる
- `W-001〜W-005` が見た目の違いだけでなく、どの制度圧がどの生態異常を作るかで区別されるようにする

## 2. 序盤の ecology design rules

| ルール | 内容 |
|--------|------|
| `body carries policy` | タブーや制度は、札痕、灰膜、刻印、欠けた香りなど、身体の二度見要素で見せる |
| `food web reveals taboo` | 何を食うかで、その土地が何を多く生んでいるかを示す |
| `human misuse exists` | 正規利用と違法利用の両方を持たせる |
| `gate state bends niche` | `Listening` は境界行動と模倣、`Awakened` は攻撃性と選別行動を強める |
| `recruit fantasy survives` | 不気味でも、仲間にしたくなる機能・性格・見た目の引っかかりを残す |

### 2.1 gate state と序盤生態の関係

| gate state | 生態への出方 | 序盤で使う例 |
|------------|--------------|--------------|
| `Listening` | 境界線、呼び声、札や灰など人の痕に寄る | 開始村, `W-001`, `W-002`, `W-003`, `W-005` |
| `Awakened` | 通行、順位、潮汐、音に反応して選別行動が強まる | `W-004` |

## 3. 開始村と塔前荒地の生態

### 3.1 基礎資源

| 資源 | 発生源 | それを食う / 利用するもの | 物語的意味 |
|------|--------|---------------------------|------------|
| すす草 | 畜舎の灰、焼印、囲いの煤 | モクケダ, ヨモギナメ | 家畜管理と焼印文化 |
| 湿墓土 | 空碑まわりの湿気、供え草 | ヨモギナメ, シメリガサ | 死が終わっていない |
| 札屑 | 家畜札、削り跡、木札片 | タグツツキ, カゴホネズミ, シルシクイ | 所属が物質として散っている |
| 井戸藻 | 井戸、洗い場、乳桶まわり | ヨモギナメ | 生活感のベース |
| 骨粉と塩 | 墓地、灰窯、保存仕事 | カゴホネズミ, シメリガサ | 弔いと保存が近い |

### 3.2 trophic web

| 層 | 主役 | 食性 / 行動 | 人間との接点 |
|----|------|-------------|--------------|
| `primary grazers` | モクケダ, ヨモギナメ | 草、藻、湿土、灰の混じる飼葉 | 家畜と野生の境界を曖昧にする |
| `tag scavengers` | タグツツキ, カゴホネズミ | 札屑、木片、布切れ、骨片 | 家や記録の痕跡を巣へ運ぶ |
| `boundary mimics` | ユビガラス, マヨイカカシ | 人の声、道標、警告杭に寄る | 噂と迷いを増幅する |
| `mark-eaters` | シルシクイ | 印、封、乾いた塗膜、札穴 | 記録や門のしるしを食う |
| `gate-resonant rare` | トウモリノコ | 境界風、記憶の強い匂い、門前の静音 | 門の近さと主人公の違和感を橋渡しする |

### 3.3 プレイヤーが拾うべきこと

- 家畜系と野生系の線が弱い
- `札` と `餌` と `供え物` が近い場所にある
- 人間の管理物がそのままモンスターの資源になっている

## 4. `W-001 名伏せの野`

### 4.1 生態の主題

- `所属の仮止め` が生態にも出る
- 鈴、札、柵といった人間の管理物が、そのまま餌場と縄張りの基準になる

### 4.2 food web

| trophic band | 主役 | 何を食う / 追う | 見える違和感 |
|--------------|------|------------------|--------------|
| `basal` | 湿草、札縄、乾草黴 | 風で集まる | 札屑が土に混じる |
| `grazers` | モクケダ, イワミミ | 灰混じりの草、崖草、塩 | 柵の内外で毛並みが変わる |
| `scavengers` | タグツツキ, カゴホネズミ | 借り名札、鈴紐、戸口の木片 | 人名札を巣材にする |
| `call mimics` | ユビガラス | 人の呼び声、鈴音 | 役割名だけまねる |
| `mark predators` | シルシクイ | 札の塗膜、祠の印 | 札穴から食い始める |
| `rare gate bound` | トウモリノコ | 門風、濡れた札、静音 | 名を返すように立ち止まる |

### 4.3 human use / abuse

| 用途 | 正規 | 違法 / 逸脱 |
|------|------|-------------|
| `シメリガサ胞子` | 乾草倉の眠り避け、夜番の落ち着き | 名を忘れさせるための混ぜ物 |
| `シルシクイ爪粉` | 古い印の剥離 | 家畜札や借り名札の削除 |
| `タグツツキ嘴片` | 木柵の彫り直し | 借り札の穴位置の偽装 |
| `鈴紐` | 個体識別 | 逃亡者の追跡補助 |

### 4.4 mutation seeds

| draft_id | 名前 | family | ontology_class | 説明 |
|----------|------|--------|----------------|------|
| `VAR-W01-01` | 濡札シルシクイ | magic | record-bent | 濡れた札から先に食い、所属だけでなく呼び名もぶらす |
| `VAR-W01-02` | 返名トウモリノコ | divine | gate-touched | 名を呼ばれた回数の多い者の前でだけ振り返る |
| `VAR-W01-03` | 鈴喰ユビガラス | bird | record-bent | 鈴の音色を奪い、同じ家の者が同音になる |

### 4.5 habitat revision proposal

- 葦原の外縁に `濡れ札の浅水域` を追加し、札が自然物でなく環境資源になっていることを見せる
- 集落と祠のあいだに `借り札の乾し棚` を置き、人の制度が生態インフラ化していると示す
