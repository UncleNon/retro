## 1. 目的

- 状態異常の全ルールを一箇所に集約し、バトル設計・AI設計・UI設計が同一の定義を参照できるようにする
- フィールド効果（天候・地形）のバトルへの影響を明確に数値化する
- 重複・競合・耐性のルールを曖昧さなく定義する

## 3. 各状態異常 詳細定義

### 3.1 POISON（毒）

| 項目 | 値 |
|------|----|
| 内部ID | `POISON` |
| 表示名 | 毒 |
| カテゴリ | 継続ダメージ |
| 効果 | ラウンド終了時に `floor(max_hp / 16)` のダメージ（最低1） |
| 基本持続 | 4ターン |
| 持続計算 | `duration = max(1, base_duration - floor(target.res / 80))` |
| 自然回復 | 持続ターン経過で自動解除 |
| アイテム治療 | 毒消し草、万能薬 |
| スキル治療 | キアリー系、浄化系 |
| 重複ルール | 再付与時、残ターンが新規付与の持続より短ければ上書き。長ければ無視 |
| 戦闘終了時 | 自動解除（フィールドに持ち越さない） |
| AI反応 | `INTELLIGENT`: 回復優先度 +8 / `FERAL`: 無視 / `STAY_SAFE`: 即時治療試行 |
| 特記 | 呪いと同時付与可。毒ダメージは防御で軽減不可 |

### 3.2 SLEEP（眠り）

| 項目 | 値 |
|------|----|
| 内部ID | `SLEEP` |
| 表示名 | 眠り |
| カテゴリ | 行動阻害 |
| 効果 | 行動不能。被ダメージ時に `35%` の確率で解除 |
| 基本持続 | 2ターン（1〜3ターンの範囲、RES依存） |
| 持続計算 | `duration = max(1, random(1, 3) - floor(target.res / 100))` |
| 自然回復 | 持続ターン経過 or 被ダメージ |
| アイテム治療 | 目覚まし草、万能薬 |
| スキル治療 | ザメハ系、浄化系 |
| 重複ルール | 麻痺と同時付与不可（後発を拒否）。凍結とも同時不可 |
| 被ダメージ解除 | `35%` 確率。会心攻撃の場合 `70%` |
| AI反応 | `INTELLIGENT`: 眠り中の敵への攻撃を避ける（残ターン1以上なら他を狙う） / `FERAL`: 構わず殴る |
| 勧誘補正 | 眠り中の敵への勧誘成功率 +10 |
| 特記 | 眠り中は回避率0。initiative計算をスキップ |

### 3.3 PARALYSIS（麻痺）

| 項目 | 値 |
|------|----|
| 内部ID | `PARALYSIS` |
| 表示名 | 麻痺 |
| カテゴリ | 行動阻害 |
| 効果 | 行動失敗率 `25%`。`initiative -8` |
| 基本持続 | 3ターン（2〜4ターンの範囲） |
| 持続計算 | `duration = max(2, random(2, 4) - floor(target.res / 120))` |
| 自然回復 | 持続ターン経過のみ |
| アイテム治療 | 痺れ消し草、万能薬 |
| スキル治療 | キアリク系、浄化系 |
| 重複ルール | 眠りと同時付与不可（後発を拒否） |
| AI反応 | `INTELLIGENT`: 麻痺中の味方に治療優先度 +10 / 敵AIは麻痺中の相手を好んで攻撃 (+6) |
| 勧誘補正 | 麻痺中の敵への勧誘成功率 +6 |
| 特記 | 麻痺で行動失敗した場合、MPは消費しない |

### 3.4 CONFUSION（混乱）

| 項目 | 値 |
|------|----|
| 内部ID | `CONFUSION` |
| 表示名 | 混乱 |
| カテゴリ | 行動変質 |
| 効果 | `33%` で自傷（味方 or 自分に通常攻撃）。`33%` でランダム対象に攻撃。`34%` で正常行動 |
| 基本持続 | 2ターン |
| 持続計算 | `duration = max(1, base_duration - floor(target.res / 100))` |
| 自然回復 | 持続ターン経過。被ダメージでは解除しない |
| アイテム治療 | 正気の実、万能薬 |
| スキル治療 | ツッコミ系、浄化系 |
| 重複ルール | 魅了と同時付与不可（後発を拒否） |
| 自傷ダメージ | 通常攻撃と同じ計算式。ただし会心は発生しない |
| AI反応 | `INTELLIGENT`: 混乱中の味方を攻撃対象から除外（同士討ち狙い） / `PACK`: 混乱した味方からかばうを解除 |
| 特記 | 混乱中に呪文は使用しない（物理攻撃のみ） |

### 3.5 SEAL（封印）

| 項目 | 値 |
|------|----|
| 内部ID | `SEAL` |
| 表示名 | 封印 |
| カテゴリ | 能力制限 |
| 効果 | 呪文カテゴリを封じる。`seal_target = magic` の場合は呪文全般、`seal_target = physical_skill` の場合は特技全般 |
| 基本持続 | 3ターン |
| 持続計算 | `duration = max(2, base_duration - floor(target.res / 100))` |
| 自然回復 | 持続ターン経過のみ |
| アイテム治療 | 封印解きの札、万能薬 |
| スキル治療 | 封印解除系（封印されていない系統の技で使用可） |
| 重複ルール | 呪文封印と特技封印は同時付与可。同カテゴリの再付与は上書き判定 |
| AI反応 | `RITUAL`: サイクルがリセットされる / `INTELLIGENT`: 封印されていない系統で攻撃を継続 / `FERAL`: 通常攻撃に切替 |
| 特記 | 封印中に封じられた系統を選択すると「しかし技が出せない！」表示。ターンは消費する |

### 3.6 FEAR（恐怖）

| 項目 | 値 |
|------|----|
| 内部ID | `FEAR` |
| 表示名 | 恐怖 |
| カテゴリ | 行動阻害 |
| 効果 | 行動キャンセル率 `20%`。`initiative -5`。逃走コマンドのスコア +15（味方AI） |
| 基本持続 | 2ターン |
| 持続計算 | `duration = max(1, base_duration - floor(target.res / 120))` |
| 自然回復 | 持続ターン経過のみ |
| アイテム治療 | 勇気の粉、万能薬 |
| スキル治療 | 鼓舞系、浄化系 |
| 重複ルール | 魅了と同時付与可。封印とも同時付与可 |
| AI反応（被害側）| `STAY_SAFE`: 防御優先度 +14 / `GO_ALL_OUT`: 恐怖を無視して攻撃（行動キャンセルは発生する） |
| AI反応（加害側）| `TERRITORIAL`: 恐怖中の相手への攻撃優先度 +8 |
| 特記 | 恐怖で行動キャンセルされた場合、「おびえて動けない！」表示。ボスは恐怖耐性が高い（通常 +1〜+2） |

### 3.7 BURN（燃焼）

| 項目 | 値 |
|------|----|
| 内部ID | `BURN` |
| 表示名 | 燃焼 |
| カテゴリ | 継続ダメージ + 弱体 |
| 効果 | ラウンド終了時に `floor(max_hp / 12)` のダメージ（最低1）。`def -10%` |
| 基本持続 | 3ターン |
| 持続計算 | `duration = max(1, base_duration - floor(target.res / 100))` |
| 自然回復 | 持続ターン経過。水属性攻撃被弾で即時解除 |
| アイテム治療 | 冷却水、万能薬 |
| スキル治療 | 水属性回復技、浄化系 |
| 重複ルール | 凍結と同時付与不可（後発が優先、先発を上書き）。毒と同時付与可 |
| 属性連携 | 燃焼中に風属性攻撃を受けると追加ダメージ `floor(max_hp / 20)` |
| AI反応 | `INTELLIGENT`: 燃焼中の敵には火属性攻撃を避ける（追加効果なし）。水属性があればそちらを使用 |
| 特記 | 火属性技で付与。雨天時は燃焼持続 -1ターン |

### 3.8 FREEZE（凍結）

| 項目 | 値 |
|------|----|
| 内部ID | `FREEZE` |
| 表示名 | 凍結 |
| カテゴリ | 行動阻害 + 脆弱 |
| 効果 | 行動不能。物理被ダメージ `+25%`。被ダメージ時に `40%` で解除 |
| 基本持続 | 1ターン（0〜2ターンの範囲） |
| 持続計算 | `duration = max(0, random(0, 2) - floor(target.res / 80))` |
| 自然回復 | 持続ターン経過 or 被ダメージ（火属性被弾で確定解除） |
| アイテム治療 | 解凍薬、万能薬 |
| スキル治療 | 火属性技（ダメージと同時に解除）、浄化系 |
| 重複ルール | 眠りと同時付与不可。燃焼と同時付与不可（後発優先で上書き） |
| 脆弱効果 | 凍結中に物理攻撃を受けると `damage * 1.25` |
| AI反応 | `INTELLIGENT`: 凍結中の敵に物理攻撃を優先 (+12 score) / `PACK.alpha`: 凍結敵への集中攻撃指示 |
| 特記 | 水属性技の一部で付与。duration 0 の場合は「凍りかけたが耐えた！」表示のみ |

### 3.9 CURSE（呪い）

| 項目 | 値 |
|------|----|
| 内部ID | `CURSE` |
| 表示名 | 呪い |
| カテゴリ | 継続ダメージ |
| 効果 | 行動実行時に `floor(max_hp / 20)` の自傷ダメージ（最低1） |
| 基本持続 | 4ターン |
| 持続計算 | `duration = max(2, base_duration - floor(target.res / 120))` |
| 自然回復 | 持続ターン経過のみ。被ダメージでは解除しない |
| アイテム治療 | 祓いの札、万能薬 |
| スキル治療 | シャナク系、光属性浄化 |
| 重複ルール | 毒と同時付与可。刻印と同時付与可 |
| 自傷タイミング | 行動宣言直後、効果解決前にダメージ。自傷で倒れた場合、行動は不発 |
| AI反応 | `STAY_SAFE`: 防御を選択（防御は行動だが自傷は `floor(max_hp / 20 * 0.50)` に軽減）/ `INTELLIGENT`: 治療優先度 +12 |
| 特記 | 闇属性技で付与。光属性耐性が高いモンスターは呪い耐性も高い傾向 |

### 3.10 BLIND（暗闇）

| 項目 | 値 |
|------|----|
| 内部ID | `BLIND` |
| 表示名 | 暗闇 |
| カテゴリ | 弱体 |
| 効果 | 物理命中 `-12`。対象の回避に `+6` ボーナス相当 |
| 基本持続 | 3ターン |
| 持続計算 | `duration = max(1, base_duration - floor(target.res / 100))` |
| 自然回復 | 持続ターン経過のみ |
| アイテム治療 | 目薬草、万能薬 |
| スキル治療 | キアリー系、浄化系 |
| 重複ルール | 他の状態異常とすべて同時付与可 |
| 命中修正詳細 | 攻撃側が暗闇: `physical_hit -12` / 防御側が暗闇: 攻撃側の `physical_hit +6` |
| AI反応 | `INTELLIGENT`: 暗闇中は呪文攻撃を優先（呪文は暗闇の影響を受けない） / `FERAL`: 通常通り物理攻撃（ミス多発） |
| 特記 | 呪文命中率には影響しない。暗闇中のフィールド移動にはペナルティなし（バトル限定効果） |

| 項目 | 値 |
|------|----|
| 内部ID | `CHARM` |
| 表示名 | 魅了 |
| カテゴリ | 行動変質 |
| 効果 | `50%` で行動不能（うっとりして動けない）。`30%` で味方を攻撃。`20%` で正常行動 |
| 基本持続 | 2ターン |
| 持続計算 | `duration = max(1, base_duration - floor(target.res / 80))` |
| 自然回復 | 持続ターン経過。味方からの攻撃被弾で `50%` 解除 |

| 基本持続 | 3ターン |

| 自然回復 | 持続ターン経過のみ |

|------|----|
| 内部ID | `DOOM` |
| 表示名 | 死の宣告 |
| カテゴリ | 遅延致死 |
| 効果 | カウントダウン終了時に即死。カウント中は他の効果なし |
| 基本持続 | 3ターン（カウントダウン） |
| 持続計算 | `countdown = max(2, base_duration - floor((caster.int - target.res) / 40))` |
| 自然回復 | カウントダウン中に術者を倒すと解除 |
| アイテム治療 | 蘇生不可。予防的に万能薬で解除可 |
| スキル治療 | ザオリク系（即死後）、発動前ならシャナク系で解除 |

## 4. 状態異常耐性

### 4.1 耐性値システム

耐性値は `-2` から `+2` の5段階。`monster_resistance.csv` で種族ごとに定義。

| 耐性値 | 表示 | 付与率への補正 |
|--------|------|---------------|
| `-2` | 弱点 | +12 |
| `-1` | やや弱い | +6 |
| `0` | ふつう | 0 |
| `+1` | 耐性 | -10 |
| `+2` | 強耐性 | -20 |

### 4.2 状態異常成功率の最終式

```text
status_hit =
  ailment_base_rate
  + floor((caster.int - target.res) * 0.15)
  + ailment_synergy_bonus
  + target_resist_step_bonus
  + phase_ailment_bonus
  + field_ailment_bonus

final_status_hit = clamp(status_hit, 5, 95)
```

### 4.3 各補正の詳細

| 補正項目 | 説明 | 値の範囲 |
|---------|------|---------|
| `ailment_base_rate` | 技ごとの基礎付与率 | 30〜85 |
| `int - res` 差 | 術者の賢さと対象の精神の差 | 効果: 差 × 0.15 |
| `ailment_synergy_bonus` | 既存状態異常との相乗効果 | 0〜+10 |
| `target_resist_step_bonus` | 対象の耐性段階による補正 | -20〜+12 |
| `phase_ailment_bonus` | ボスフェーズによる補正 | -35〜0 |
| `field_ailment_bonus` | フィールド効果による補正 | -10〜+10 |

### 4.4 相乗効果（Synergy）テーブル

| 既存状態 | 新規付与 | synergy_bonus |
|---------|---------|--------------|
| 毒 | 呪い | +5 |
| 燃焼 | 恐怖 | +4 |
| 麻痺 | 刻印 | +6 |
| 暗闇 | 混乱 | +4 |
| 封印 | 沈黙 | +8 |

## 5. 状態異常の重複・競合ルール

### 5.1 排他グループ

| グループ | 含まれる状態 | ルール |
|---------|------------|--------|
| A: 意識喪失 | 眠り, 凍結 | 同時付与不可。後発を拒否 |
| B: 行動変質 | 混乱, 魅了 | 同時付与不可。後発を拒否 |
| C: 温度 | 燃焼, 凍結 | 同時付与不可。後発が優先（先発を上書き） |

### 5.2 同時付与可能な組み合わせ例

- 毒 + 呪い + 暗闇（3重苦）
- 麻痺 + 封印 + 刻印（行動制限コンボ）
- 恐怖 + 封印 + 沈黙（完全封殺）
- 毒 + 燃焼 + 刻印（継続ダメージ極大）

### 5.3 付与上限

```text
max_concurrent_ailments = 3

if target already has 3 ailments:
  new ailment is rejected
  message: "しかし これ以上は効かなかった！"

exception:
  刻印(MARK) は付与上限にカウントしない（デバフ扱い）
  死の宣告(DOOM) は付与上限にカウントしない
```

### 5.4 再付与ルール

```text
same_ailment_reapply:
  if new_duration > remaining_duration:
    overwrite with new_duration
  else:
    reject (message: "しかし すでに かかっている！")

  exception: DOOM は常に短い方を採用
```

| `TERRITORIAL` | 恐怖付与で侵入者を排除 |
| `INTELLIGENT` | 弱点に合わせて最適な状態異常を選択 |
| `RITUAL` | サイクルの一部として計画的に使用 |
| `AMBUSH` | 初手で眠り or 麻痺を狙う（時間稼ぎ） |
| `GATE_GUARDIAN` | ギミックに関連する状態異常を使用 |
| `ELDER_BOSS` | フェーズごとに異なる状態異常戦略 |

## 11. テレメトリ連携

### 11.1 状態異常関連テレメトリ

| event | フィールド |
|-------|----------|
| `ailment_applied` | `source_id`, `target_id`, `ailment_id`, `duration`, `hit_rate`, `resist_step`, `field_effect` |
| `ailment_cured` | `target_id`, `ailment_id`, `cure_method` (natural/item/skill/damage), `remaining_turns` |
| `ailment_rejected` | `source_id`, `target_id`, `ailment_id`, `reject_reason` (resist/conflict/cap), `hit_rate` |
| `field_effect_active` | `zone_id`, `field_effect_id`, `source` (zone/skill/boss), `round_no` |
| `hazard_damage` | `zone_id`, `hazard_id`, `damage`, `step_count`, `mitigation` |

### 11.2 アラート条件

- 特定状態異常の成功率が想定より `+15pt` 以上乖離 → 耐性テーブル調整必要
- 毒 + 呪い + 燃焼の3重継続ダメージで1ラウンドにHP `30%` 以上減少 → 継続ダメージ過剰
- 万能薬の使用率がアイテム全体の `40%` を超える → 個別治療アイテムの入手性不足
- ボス戦での状態異常付与成功率が `10%` 未満 → ボス耐性が高すぎる
- 特定の天候でのプレイヤー勝率が通常より `15%` 以上低下 → 天候補正が強すぎる
- フィールドハザードでの全滅率が `5%` を超える → ハザードダメージ過剰 or 軽減手段不足
