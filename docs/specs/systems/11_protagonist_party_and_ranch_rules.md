### 2.3 初期所持モンスター

**判定: 半家畜状態のモンスター1体を持たせる**

| 項目 | 値 |
|------|----|
| 初期個体 | `MON-001` モクケダ |
| 名前 | プレイヤーが付ける（デフォルト提案: ケダ） |
| レベル | 3 |

- 「勧誘で0→1体にする」よりも「1体いる状態で2体目を迎える」方が導線が自然
- 配合で「仕事仲間を素材にするか」という不可逆判断が早期に来る

### 3.2 衣装

## 4. パーティ枠

### 4.1 構成

| 枠 | 数 | 備考 |
|----|----|------|
| 戦闘参加枠 | 3 | 常に3体が戦闘に出る |
| 控え枠 | 0 | 戦闘中の入替は「さくせん」内の交代ではなく、パーティ編成で事前に決める |

### 4.2 世界観上の理由付け

- 塔の門は「3つまでの所属印を同時に通す」という設定
- 4体目を連れると門の反応が不安定になる、という世界ルール
- これにより「誰を連れるか」の判断圧が常に発生する

### 4.3 戦闘中の入替

- **戦闘中のモンスター入替は原則不可**
- 理由: 準備（事前編成）で勝つ設計思想を守る
- 例外: 特定の trait `非常入替` を持つ個体のみ、1戦に1回だけ牧場から呼べる（endgame 向け）

## 5. 牧場

### 5.1 基本仕様

| 項目 | 値 |
|------|----|
| 最大預かり数 | 38 |
| アクセス場所 | 村の畜舎 / 各世界の拠点牧場 / 塔内部の一時預かり所 |
| フィールドからの直接アクセス | 不可 |
| パーティ ↔ 牧場の入替 | 拠点でのみ可能 |

### 5.2 38体の根拠

- パーティ3体 + 牧場38体 = 合計41体
- 序盤は余裕があるが、中盤で配合素材と戦力保持が衝突し始める
- 終盤では「誰を残し、誰を配合に回すか」が常に判断対象になる
- 200体預かりにすると圧が消える。38体は DQM1 の牧場圧に近い水準

### 5.3 牧場内の変化

| 項目 | 変化 |
|------|------|
| 忠誠度 | 預けている間は変化なし（減りもしない） |
| HP/MP | 預けると全回復 |
| 気質 | 変化なし |
| 成長 | 預けている間は経験値が入らない |
| 休息ボーナス | なし。シンプルに保つ |

### 5.4 お気に入りロック

| 項目 | 値 |
|------|----|
| ロック枠 | 5 |
| ロック効果 | 配合親候補から除外、売却不可、リリース不可 |
| 解除 | いつでも可能 |

### 5.5 リリース

- 牧場が満杯のとき、新規勧誘 or 配合結果を受け取れない
- リリースすると完全に消える（取り消し不可）
- リリース前に確認ダイアログを2回出す
- お気に入りロック中の個体はリリース不可

## 6. 忠誠度 / なつき度

### 6.1 内部値

| 項目 | 値 |
|------|----|
| 初期値 | 勧誘: 30 / 配合子: 60 / 初期モクケダ: 80 |
| 最大値 | 100 |
| 最小値 | 0 |

### 6.2 変化条件

| 条件 | 変化量 |
|------|-------:|
| 戦闘勝利 | +1 |
| 作戦通りの行動 | +1 |
| HP0 から復帰（蘇生） | -3 |
| 敗北時に生存 | -1 |
| 全滅 | -5 |
| 配合の親に選ばれた（消滅前） | ±0 |
| 牧場に長期放置（100戦以上） | ±0（変化なし） |
| 好物の bait を与えた（戦闘外） | +2 |
| 嫌いな bait を与えた | -1 |

### 6.3 忠誠度の効果

| 忠誠度帯 | 効果 |
|----------|------|
| 80〜100 | 作戦 AI の精度が最高。自動で最適行動を選びやすい |
| 60〜79 | 通常。作戦通り動く |
| 40〜59 | たまに作戦を無視する（10%の確率で別行動） |
| 20〜39 | 作戦無視が20%に増える。命令させろ が不安定になる |
| 0〜19 | 作戦無視が35%。逃亡を試みることがある |

### 6.4 表示方針

- 数値を直接見せない
- 「よくなついている」「すこし不安そうだ」「落ち着きがない」など文言で返す
- 牧場画面のアイコン（目の表情差分: 3段階）で補助

## 7. 性格システム

### 7.1 性格の内部構造

DQM1 の 3軸×3段階 = 27種を参考に、本作では **2軸×3段階 = 9種** に圧縮する。

| 軸 | 高 | 中 | 低 |
|----|----|----|-----|
| **積極性** (aggression) | 猛 | 普 | 穏 |
| **慎重さ** (prudence) | 知 | 普 | 荒 |

### 7.2 9種の性格

| 性格名 | aggression | prudence | AI傾向 |
|--------|------------|----------|--------|
| 猛知 | 高 | 高 | 弱点を突いて攻める |
| 猛普 | 高 | 中 | ひたすら攻撃 |
| 猛荒 | 高 | 低 | 全力攻撃、MP無視 |
| 普知 | 中 | 高 | バランス重視、補助を使う |
| 普普 | 中 | 中 | 標準AI |
| 普荒 | 中 | 低 | やや攻撃寄り、回復軽視 |
| 穏知 | 低 | 高 | 補助・回復優先 |
| 穏普 | 低 | 中 | 様子見が多い |
| 穏荒 | 低 | 低 | 不規則、予測しにくい |

### 7.3 性格の決定

- 勧誘時: モンスター種ごとの `personality_bias` + ランダム
- 配合時: 両親の性格値を平均 + ランダム微調整
- 変化条件: なし（固定）。性格変更は配合でのみ行う

### 7.4 性格補正

| 性格 | 成長補正 | 戦闘補正 |
|------|----------|----------|
| 猛系 | ATK +3%, DEF -2% | 攻撃技優先度 +10 |
| 穏系 | DEF +3%, ATK -2% | 回復/補助優先度 +10 |
| 知系 | INT +3%, HP -2% | 弱点検知、MP管理 |
| 荒系 | SPD +3%, RES -2% | ランダム行動混入 |
| 普系 | 補正なし | 補正なし |

## 8. 戦闘不能のペナルティ

### 8.1 通常戦闘

| 状態 | ペナルティ |
|------|------------|
| HP0 | 戦闘後に HP 1 で復帰 |
| 忠誠度 | -3 |
| 戦闘報酬 | 正常に受け取れる |

### 8.2 全滅

| 状態 | ペナルティ |
|------|------------|
| 全滅時 | 所持金10%ロスト、最後の拠点に戻る |
| HP/MP | 全員 HP 1 / MP 0 で復帰 |
| 忠誠度 | 全員 -5 |
| アイテム | 失わない |

### 8.3 長期的ペナルティ

- 戦闘不能そのものに長期ペナルティは **設けない**
- 理由: 忠誠度低下で十分。やる気を折るペナルティはレトロ体験でも不要
- 老成/覚醒イベントも **設けない**。配合による世代交代が本作の主軸であり、1体を永久運用するゲームではない

## 9. 主人公の年齢が与える差分

### 9.1 NPC反応パターン

| NPC層 | 主人公への態度 |
|-------|----------------|
| 若者NPC | 敬語を使う。おじさん/おばさん呼びで距離がある |
| 同年代NPC | 名前で呼ぶ。対等な会話。昔話を共有する |
| 長老NPC | 「お前もいい歳だ」「もう分かっているだろう」と含みを持たせる |
| 子どもNPC | 無邪気に話す。禁忌を知らない |
| 他世界のNPC | 年齢を見て「旅慣れている」と推測する |

### 9.2 選択肢への影響

- 若者主人公では出せない選択肢を、45歳だからこそ出せる
- 例: 「……知っていた」「昔からおかしいと思っていた」「もう黙っていられない」
- 少年RPGの「驚き」ではなく、大人の「確認」と「決断」をトーンにする

## 10. QA Checklist

- 初期所持品で最初の遭遇を乗り切れるか
- 初期モクケダのLv3が序盤戦闘バランスに合っているか
- 牧場38体で中盤の配合圧が発生するか
- 忠誠度が体感上意味を持つか（低忠誠のデメリットが見えるか）
- 性格9種が AI の差として実装可能か
- 見た目差分が 16×16 で区別可能か
