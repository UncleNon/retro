## 3.1 世界観コンセプト

### トーン: ダークファンタジー
> 表面は色鮮やかなドット絵のファンタジー世界。しかしプレイヤーが世界を深く知るほど、その裏にある闇——禁忌の歴史、人間の欲望、モンスターの真の起源——が浮かび上がる。

### 世界観の三層構造

#### 第一層: 表の世界（プレイヤーが最初に見るもの）
- 美しいファンタジー世界
- 人とモンスターが共存する牧歌的な雰囲気
- 冒険者ギルド、トーナメント、牧場経営
- 「楽しいモンスター育成RPG」として成立する層

#### 第二層: 裏の世界（ストーリー中盤で明かされる）
- モンスターの「配合」という行為の倫理的問題
- 世界間の行き来が生む歪み
- 各世界の支配構造と搾取の歴史
- NPCたちの裏の顔、二面性
- 「この世界、思ったより闇深いな」と感じる層

#### 第三層: 深淵（エンドコンテンツで到達）
- モンスターとは何者なのか、の根源的な答え
- 世界が20以上に分かれている本当の理由
- 主人公が45歳である必然性と結びつく物語
- 「終わったと思ったらまだあった」の連鎖
- プレイヤーの価値観を揺さぶる展開

## 3.2 世界構造

### 20+の世界
各世界は独立した文化圏・生態系・歴史を持つ。世界間の移動は、中央ハブに並ぶ「生きた門」を通じて行う。門は単なる装置ではなく、世界同士の継ぎ目に根を張る存在であり、特定の条件（ストーリー進行、キーアイテム、トーナメントランク、系統の共鳴）で反応して開く。

### 世界間移動の基盤: 中央ハブ + 生きた門
- 中央ハブの仮称は **塔**。世界間移動の起点であり、物語とUIの両方の中心となる
- 塔は、聖堂、修道院、寺社、天文台、墓所のどれとも断定できない「意図不明の建物」として設計する
- 塔は世界の外に浮かぶ抽象空間ではなく、山、谷、海、町、城、洞窟がある通常のファンタジー地形の中に、異物として存在する一建築物である
- 各世界への入口は「生きた門」。古代の巨大モンスターの遺骸、器官、殻、骨格などを思わせる不気味な造形を持つ
- 門は世界ごとに見た目と反応条件が異なる。火山世界の門、鏡世界の門、禁忌世界の門では開き方自体が違う
- 世界移動そのものを「儀式」「侵入」「越境」と感じさせ、単なるメニュー遷移にはしない
- 主人公とモンスターの関係、世界の分裂、配合の倫理がこの移動構造と一本でつながるよう設計する
- 「誰が何のために作ったのか」は簡単に説明せず、上位存在が生命や世界をこね回した痕跡に見える程度の匂わせで保つ

### 世界設計テンプレート
各世界は以下の要素を定義する:

```yaml
world:
  id: W-001
  name_jp: "（世界名）"
  name_en: "（World Name）"
  theme: "（テーマ: 火山、氷原、森林、都市、海底 等）"
  culture:
    political_system: "（政治体制）"
    religion: "（信仰・宗教）"
    economy: "（経済構造）"
    taboo: "（禁忌・タブー）"
    dark_secret: "（裏の闇）"
  ecology:
    dominant_species: "（支配的なモンスター種族）"
    unique_monsters: ["（固有モンスターID）"]
    environmental_hazard: "（環境的脅威）"
  history:
    founding: "（成り立ち）"
    major_events: ["（歴史的大事件）"]
    connection_to_other_worlds: "（他世界との関係）"
  gameplay:
    dungeons: ["（ダンジョンリスト）"]
    hub_town: "（拠点の町）"
    key_npcs: ["（重要NPC）"]
    unlock_condition: "（解放条件）"
    level_range: "（推奨レベル帯）"
    boss: "（ボスモンスター）"
```

### 世界間の関係性
- 世界同士は孤立していない。歴史的に繋がっている
- 「世界Aで起きた事件の余波が世界Bの文化を形成した」等の因果関係
- 一部の世界は対になっている（光と闇、火と水 等）
- 世界の数が20以上ある理由自体がストーリーの核心に関わる
- 塔と生きた門は、その「繋がってしまっている世界」の傷跡であり、意味が失われた神話遺物のように扱われる

## 3.3 歴史設計

### タイムライン（大枠）

全世界に共通する歴史的節目を設定し、各世界がそれぞれの立場からその事件を記録している。同じ事件でも世界によって伝承が異なる（信頼性の揺らぎ）。

```
■ 創世期（〜3000年前）
  - 世界の創造。モンスターの起源。
  - この時代の記録はほぼ神話として伝わる
  - 各世界で「創造神話」が異なる → 真実は？

■ 古代（3000年前〜1000年前）
  - 世界がまだ一つだった（あるいは少なかった）時代
  - 「大分裂」の原因となる事件
  - 古代文明の遺跡が各世界に残る

■ 大分裂（約1000年前）
  - 世界が複数に分かれた決定的事件
  - 各世界で異なる伝承として残る
  - この事件の真相がストーリーの核心の一つ

■ 中世（1000年前〜500年前）
  - 分裂した各世界が独自の文明を発展
  - 世界間交流の断絶と再開
  - モンスターと人間の関係が世界ごとに分岐

■ 近世（500年前〜100年前）
  - 「配合」技術の発見/再発見
  - 配合がもたらした恩恵と禍
  - 世界間の政治的緊張

■ 近代（100年前〜現在）
  - トーナメント文化の成立
  - 主人公の生きる時代
  - 新たな危機の兆候
```

### 歴史設計の原則
1. **全ての歴史的事件にモンスターが関わる**: モンスターは自然災害のような存在でも、兵器でもあり得る
2. **信頼できない語り手**: 各世界の歴史書は自国に都合よく書かれている
3. **プレイヤーが断片を集めて真実を組み立てる**: 一箇所では全体像が見えない
4. **歴史の闇が現在のクエストに直結**: 過去の禁忌が今の問題を引き起こしている

## 3.4 モンスターの存在論

### モンスターとは何か
この問いへの答えは段階的に明かされる:

#### 表向きの認識（序盤）
- 自然界に生息する生物の一種
- 人間と共存可能で、スカウト・育成できる
- 配合で新種を生み出せる不思議な生き物

#### 裏の真実（中盤以降）
- 配合がなぜ可能なのか？ 通常の生物では説明がつかない
- モンスターの「魂」「意識」はどうなっているのか
- 一部のモンスターは明らかに人工的な存在
- 世界の分裂とモンスターの多様性に相関がある
- 塔と生きた門は、人間の技術や信仰だけでは説明できない

#### 深淵の真実（エンドコンテンツ）
- （ストーリー詳細設計時に確定。ここではプレースホルダー）
- モンスターの起源は世界の根幹に関わる
- 「配合」という行為の本当の意味
- 主人公がモンスターマスターである必然性
- 生きた門がなぜ存在し、なぜモンスターの系統や変異と共鳴するのか
- 上位の何かが生命や世界をいじった痕跡なのか、それとも人間側の解釈に過ぎないのか

### 人間とモンスターの関係（世界ごとの多様性）
| パターン | 例 |
|----------|-----|
| **共存型** | 人とモンスターが対等なパートナー |
| **支配型** | 人がモンスターを道具として利用 |
| **恐怖型** | モンスターが人間社会の脅威 |
| **信仰型** | モンスターを神として崇拝 |
| **禁忌型** | モンスターとの接触自体がタブー |
| **融合型** | 人とモンスターの境界が曖昧 |

## 3.5 主人公設計

### 基本設定
| 項目 | 仕様 |
|------|------|
| **年齢** | 45歳 |
| **性別** | 選択制（男性/女性） |
| **タイプ** | 無言主人公 |
| **外見** | 選択制（ドット絵でのバリエーション） |
| **立場** | 物語開始時点でその世界の住人 |
| **生業** | 家畜番 |
| **開始地点** | 小さな村 |
| **導入動機** | 主人公が自分から塔へ向かう |
| **塔への関心** | 昔から気になっていた |
| **塔の扱い** | 村では昔から近づくなと言われてきた |
| **禁忌の根拠** | 実際に被害が出ている |
| **被害の性質** | 失踪 |
| **失踪者の範囲** | 村人 |
| **失踪の扱い** | 昔の出来事として今も語り継がれている |

### 主人公が45歳である意味
- 少年少女の「何も知らない冒険の始まり」ではない
- **人生経験を積んだ大人が、それでもなお冒険に出る理由**がある
- 世界のダークな側面を理解する成熟した視点
- 過去（20年前、30年前）に何かがあった → ストーリーの深み
- 若い頃から抱え続けてきた塔への違和感や関心を、ようやく自分の意思で確かめに行ける
- 「行くな」と言われ続けた場所へ、自分の意思で向かう反抗と執着を持てる
- 禁忌が迷信ではなく現実の被害に根ざしているため、好奇心だけではない重みを持たせられる
- 被害が「死」ではなく「失踪」であることで、説明不能さと想像の余地を強く残せる
- 失踪者を村人にすることで、塔への禁忌が個人の事件ではなく共同体の記憶として根付く
- 被害が現在進行形ではなく語り継がれた過去であることで、静かな不気味さを維持できる
- 無言でも選択肢や態度で、若者主人公には出せない含意を持たせられる

### 主人公の物語的機能
- プレイヤーの分身でありつつ、固有のバックストーリーを持つ
- 塔の外で日常を生きてきた「現地の大人」として、普通の生活圏と異物の両方を知る視点を持つ
- 家畜化された生き物やモンスターの世話をしてきた人間として、命を管理する現実と愛着の両方を知っている
- 最初の一歩を他人に強制されるのではなく、自分で選んで踏み出すことで、大人の意思を持つ主人公として立たせる
- 塔への関心は最近生えたものではなく、長年棚上げしてきた違和感として持たせる
- 村の共同体が共有してきた「塔に近づくな」という空気を知った上で、それでも踏み込む人物として描く
- 村人たちは塔をただ恐れているのではなく、過去の実被害を記憶している集団として描く
- 失踪は主人公個人だけの傷ではなく、村の誰もが知る「触れてはいけない記憶」として扱う
- 噂は風化せず残っているが、誰も真相を確かめに行かないまま日常に埋め込まれている状態を作る
- 世界の闇に対して「見て見ぬふりをしない」大人としてのスタンスを行動で示す
- モンスターとの関係に対する倫理的葛藤は、選択肢・演出・周囲の反応で表現する
- 物語序盤では、モンスターを本心から愛しているのか、仕事として世話してきただけなのかを断定しない

### 家畜番であることの意味
- 戦士でも学者でもなく、生活の延長としてモンスターと関わってきた大人にできる
- 「命を世話する」「数として扱う」「情が移る」が同時に成立するため、配合の倫理と強く結びつく
- 牧歌的な日常からダークな真相へ落差を作りやすい
- プレイヤーにとっても、最初から英雄ではない地に足のついた導入になる
- 主人公のモンスター観は、台詞で早々に説明せず、行動や選択の積み重ねで見えてくる構成にする
- 小さな村から始めることで、狭い共同体の生活感と、塔の異物感の対比を強く出せる

### 無言主人公としての演出方針
- 主人公は長文を喋らない
- 意思表示は選択肢、短い相槌、表情差分、間で表現する
- NPC側が主人公の年齢や雰囲気に反応し、会話の重みを作る
- 無言だからこそ、世界の不気味さやプレイヤーの想像余地を残す

## 3.6 ストーリー設計方針

### 構造: 多層開示型
```
メインストーリー（表層）
  → 世界を渡り歩き、各世界の問題を解決する冒険譚
  → トーナメントを勝ち上がる成長物語
  → Initial Release 時点で物語として完結する

サブストーリー（中層）
  → 各世界の闇、NPCの裏事情、歴史の断片
  → メインをクリアしなくても楽しめるが、真実に近づくヒント

裏ストーリー（深層）
  → エンドコンテンツで初めてアクセスできる
  → メイン+サブで張った深層側の伏線が回収される
  → 「この世界の本当の姿」が明かされる
```

### 完結性の原則
- Initial Release のメインストーリーは、その時点で一本の作品として完結している必要がある
- 大きな謎や深層設定は残してよいが、「未完だから続き待ち」にはしない
- 裏ストーリーは本編の穴埋めではなく、完結した本編の解釈をさらに深める追加層として設計する

### 伏線設計の原則
1. **序盤の何気ないNPCの台詞が終盤で意味を持つ**
2. **各世界で矛盾する伝承 → 真実は一つ**
3. **モンスター図鑑の説明文に隠された手がかり**
4. **特定の配合でしか生まれないモンスターの存在自体が伏線**
5. **裏ストーリーをクリアすると、メインストーリーの見え方が変わる**

### テキスト設計方針
- 安易なセリフを書かない。一つ一つのテキストに意図を持たせる
- 「言わないこと」で語る（行間を読ませる）
- NPCは生活感のある台詞（世界に実在感を持たせる）
- ダークな要素は直接的に描写せず、示唆する（プレイヤーの想像力を刺激）

## 3.7 テーマ・モチーフ

### メインテーマ（候補）
- **継承と断絶**: 配合（＝命の継承）と世界の分裂（＝断絶）
- **共存の代償**: モンスターとの共存には何が犠牲になっているのか
- **大人の冒険**: 全てを知った上でそれでも進む覚悟

### 禁忌・タブー（世界観に埋め込む要素）
- 禁じられた配合（倫理的に許されない組み合わせ）
- 世界間の禁忌（越えてはならない境界がある理由）
- 歴史から消された事件（各世界が隠している過去）
- モンスターの意識に関する問い（配合で消える親の魂は？）
- 生きた門を私的に開くこと、門の素材を切り出すこと、自分の身体を門に同調させること

## 3.8 NPC設計方針

### NPC分類
| 種類 | 役割 | 例 |
|------|------|-----|
| **キーNPC** | ストーリー進行に不可欠 | 各世界のリーダー、師匠的存在 |
| **サブNPC** | サブクエスト提供、世界観補強 | 住民、商人、旅人 |
| **対戦NPC** | トーナメント対戦相手、ライバル | 他のモンスターマスター |
| **闇NPC** | 裏ストーリーに関わる | 真実を知る者、黒幕 |

### NPC設計テンプレート
```yaml
npc:
  id: NPC-001
  name_jp: ""
  name_en: ""
  world: "所属世界"
  role: "表向きの役割"
  dark_side: "裏の顔・秘密"
  relationship_to_main_story: ""
  key_dialogue_themes: []
  quest_provided: []
```

### NPC設計の原則
- **全てのキーNPCに裏がある**（善人に見えても何かを隠している）
- **サブNPCにも生活がある**（定型文の繰り返しを避ける）
- **対戦NPCは配合哲学を持つ**（「力こそ全て」「バランスが大事」等、プレイヤーに戦略のヒントを与える）

## 3.9 物語設計哲学（抽象 → 具体）

| 抽象原則 | 具体ルール |
|----------|-----------|
| **説明より先に違和感を積む** | 導入では真相を語らず、塔、村、図鑑、記録、台詞の小さな歪みを先に置く |
| **謎は必ず人間の生活に接続する** | 世界の秘密は神話だけでなく、戸籍、相続、家畜管理、葬送、結婚、村の噂に落とす |
| **不可思議さと生々しさを同時に持たせる** | 塔や門は超常的でも、そこに巻き込まれるのは具体的な村人、制度、生活苦である |
| **伏線は一箇所に集めず、ゲーム全体に散らす** | 図鑑文、配合結果、UI表現、地形、NPC会話、トーナメント、台帳で別角度から同じ真相を示す |
| **プレイヤーに“自分で気づいた”感覚を与える** | 露骨な答え合わせは避け、複数の断片が結びついたときに初めて全体像が見える構造にする |
| **メインストーリーは制度と人間の罪まで決着させる** | 初回リリース内で、失踪と塔を巡る人為的な隠蔽・利用構造には決着をつける |
| **深層設定は残すが、未完にはしない** | 上位存在や世界そのものの正体はエンドコンテンツで深掘りし、本編では“人間が何をしたか”までは明かす |
| **主人公は選ばれし者ではなく、見て見ぬふりをやめた大人** | 血統や予言ではなく、長年の違和感と意志によって塔へ踏み込む |

## 3.10 導入プロット骨子

### オープニングの基本構造
1. 小さな村で、家畜番としてのいつもの朝が描かれる
2. 主人公は家畜の数合わせと札の整理をしている最中、古い **家畜札** に不自然な削り跡と、失踪した村人の名残を見つける
3. その札に刻まれた家印や書き換え跡が、村の古い記録の修正痕と一致していることが示唆される
4. 村人は「昔のことだ」「塔に近づくな」としか言わず、真相に触れようとしない
5. 主人公は他人に背を押されるのではなく、自分で塔へ向かう
6. 塔の最初の門は、主人公が持ち込んだ札や家印に反応するように開き、最初の異世界への越境が始まる

### 導入でプレイヤーに与える印象
- 塔はただ怖いのではなく、村の生活と見えないところで繋がっている
- 失踪事件は昔話ではあるが、完全に終わった話ではない
- 主人公は英雄ではなく、生活者として違和感を放置できなくなった大人
- 物語の謎は「塔の正体」だけでなく、「村が何を知っていて黙っているか」にもある

## 3.11 失踪事件の共通項

### 表向きの説明
- 村では「塔に近づいた者が帰らなかった」と語られている
- ただし詳細は代ごとに曖昧化され、迷信、事故、蒸発、家出として言い換えられてきた

### 実際の共通項
| 項目 | 内容 |
|------|------|
| **名前の揺らぎ** | 失踪した村人は全員、戸籍や家系図に修正痕がある。養子、再婚、家の乗り換え、死んだ子の名の継承、父名不詳、使用人から家人化など |
| **所属の揺らぎ** | 生まれた家と現在の家が一致しない、あるいは村の中で“本来そこにいるはずではない人”として扱われていた |
| **生活上の役割** | 出産、葬送、相続、家畜の数合わせ、家印の付け替えなど、「命や所属を数え直す仕事」に近い場面を担っていた |
| **残留物** | 失踪者の家には、削られた名札、塗りつぶされた家印、修正された台帳、左右の合わない印章などが残る |
| **時期の偏り** | 失踪年は散発的に見えるが、塔の門が強く反応した年と重なっている |

### 真相に向けた設計意図
- 塔が反応するのは「特別な血」ではなく、**血筋・名前・所属のどれかがねじれている人間** だと後から見えてくる
- これは超能力的な才能ではなく、人間社会が作った帳尻合わせの傷と、世界の継ぎ目が共鳴している構造とする
- 主人公自身も完全に例外ではなく、自分の村の記録を辿るほど他人事ではなくなっていく

## 3.12 メインストーリー骨子

### Act 1: 村と塔（世界1〜3）
- 小さな村と塔周辺で導入
- 塔に入った主人公は、村の失踪者の名に似た断片を門や異世界側で見つけ始める
- 最初の数世界で、「塔に近づくな」という言い回しが世界ごとに別の形で存在することを知る
- プレイヤーは「自分の村だけの怪談ではない」と理解する

### Act 2: 反復する禁忌（世界4〜8）
- トーナメントや公的な許可がないと深い門へ進めない構造が見えてくる
- 各世界で、失踪、養子、相続、身分の付け替え、宗教的な名の変更が塔の禁忌と結びついている
- 主人公は、失踪が無差別ではなく、**書き換えられた所属を持つ者** に偏っていると気づき始める
- この時点では、敵はカルトや密輸組織のように見える

### Act 3: 間違った答え（世界9〜13）
- 塔を利用する宗教勢力、記録改竄を担う役人、門の秘密を商売にする者たちが現れる
- プレイヤーは一度、「誰かが生贄を選んで塔へ送っている」という分かりやすい陰謀論に辿り着く
- 実際にそれを利用している人間はいるが、それは真相の一部に過ぎない
- ここで主人公の村の古い記録にも、同じ改竄様式があることが明らかになる

### Act 4: 世界と名前の継ぎ目（世界14〜18）
- 塔の深部では、失踪者が単純に死んだのではなく、門や特定モンスター系統の奥に **“残響”** として取り残されていることが分かる
- 配合で生まれる一部のモンスターが、失踪者の癖、口調、記憶の断片に似た反応を見せる
- 古代の人間たちは、世界の継ぎ目を安定させるために、血筋・名前・所属のねじれた人間を **“都合の良い継ぎ木”** として扱っていたと推定される
- 現代の制度や信仰は、その残骸を半ば無自覚に再演している

### Act 5: 本編の決着（世界19〜20）
- 主人公は、塔を維持する現行の隠蔽構造と対決する
- 最終局面では「人を継ぎ目に使う」古い方法を拒否し、モンスターの血統と失踪者の名残を結び直す別の手段を選ぶ
- これにより、今後の失踪を止めること、村人たちの記憶を“なかったこと”にしないこと、塔を人食いの禁忌から外すことがメインストーリーの決着となる
- ただし、なぜ世界がそもそもこういう構造なのか、上位存在の痕跡が何なのかはエンドコンテンツ側へ残す

### メインストーリーの完結条件
- 主人公の村で起きた失踪の意味が説明できる
- 失踪を支えていた制度と隠蔽が止まる
- 主人公が塔へ向かった理由に答えが出る
- プレイヤーが「この世界の人間が何をしていたか」は理解できる
- それでもなお「世界そのものは何なのか」という余白は残る

## 3.13 主要勢力と対立構造

| 勢力 | 表向き | 裏の役割 |
|------|--------|----------|
| **村の長老たち** | 昔話を語る保守的な共同体 | 失踪を知っているが、生活を守るために黙ってきた |
| **塔の管理組織** | 世界間移動と門の保守 | 門の反応条件と失踪の記録を握り、深層を秘匿している |
| **トーナメント運営** | 実力を競う公的イベント | 門に適応できる人間とモンスターを選別する装置でもある |
| **記録官・系譜管理者** | 戸籍、相続、家印、家系図の整理 | 名前と所属の改竄を通じて、都合の悪い真実を埋める |
| **門信仰の宗派** | 塔を神聖視する宗教勢力 | 失踪を“選ばれた者の昇天”として正当化する |
| **放浪マスター / 密輸業者** | 世界を渡る異端者 | 門の危険と取引価値を知り、真実の断片を持つ |

### 対立の芯
- **善と悪の単純対立にはしない**
- 多くの勢力は「誰かを守るため」「世界が壊れないため」という理屈で沈黙や改竄を選んでいる
- プレイヤーは悪人を倒すだけでなく、**“必要悪”として回っていた構造そのもの** を理解し、壊す必要がある
