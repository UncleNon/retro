> **ステータス**: Draft v1.0
> **最終更新**: 2026-03-15

## 6.1 エンドコンテンツ設計方針

> メインクリア（60〜80時間）はゲームの半分に過ぎない。エンドコンテンツこそが、500〜1000時間のやり込みを支える核心。「終わったと思ったらまだある」を何層にも重ねる。

### Initial Release での位置づけ
- 裏ボス、裏ダンジョン、無限ダンジョンは Initial Release に含める
- エンドコンテンツは後日追加の飾りではなく、本作の価値を成立させる本体要素として扱う
- MVP では縮小検証してよいが、本番版では必須
- ただし、裏ストーリーは「本編未完の補完」ではなく、完結した本編の深掘りとして位置付ける

### プレイヤー到達マップ
```
メインストーリークリア（60〜80時間）
    ↓
裏ストーリー解放（+30〜50時間）
    ↓
裏ダンジョン群（+50〜100時間）
    ↓
高難度トーナメント（+50〜100時間）
    ↓
無限ダンジョン周回（+100〜300時間）
    ↓
図鑑コンプ・変異種収集（+100〜300時間）
    ↓
完全制覇（500〜1000時間）
```

## 6.2 裏ストーリー

### 概要
- メインクリア後に解放される追加シナリオ
- メインストーリーで散りばめた伏線が回収される
- 「世界の本当の姿」「モンスターの起源」が明かされる
- 新しい隠し世界へのアクセスが解放

### 設計方針
- メインストーリーの「もう一つの視点」として機能
- メインでは語られなかったNPCの真意が判明
- 裏ストーリーをクリアすると、メインの序盤の出来事の意味が変わる
- プレイヤーに「もう一度最初からやりたい」と思わせる

## 6.3 裏ボス

### 設計方針
- メインのラスボスとは比較にならない強さ
- 配合を極めたパーティでなければ勝てない
- 複数形態変化あり
- 勝利すると特殊配合レシピ・称号等を入手

### 裏ボス候補カテゴリ
| カテゴリ | 説明 |
|----------|------|
| **各世界の真のボス** | メインでは対面しなかった各世界の最強存在 |
| **歴史上の存在** | 1000年前の「大分裂」に関わった存在 |
| **メタ的存在** | 世界の外側から来た存在 |
| **条件付きボス** | 特定の配合・特定のアイテム・特定の行動履歴で出現 |

## 6.4 裏ダンジョン群

### 種類
| ダンジョン | 特徴 | 解放条件 |
|-----------|------|---------|
| **各世界の裏ダンジョン** | 本編では行けなかった深層 | メインクリア後 |
| **試練の塔** | 100Fの固定ダンジョン | 裏ストーリー進行 |
| **深淵の回廊** | 無限ダンジョン（メイン） | Aランクトーナメント制覇 |
| **禁忌の遺跡** | 配合禁止・持ち込み制限 | 特殊条件 |
| **時空の裂け目** | 過去の世界を探索 | 裏ストーリークリア後 |

## 6.5 高難度トーナメント

### エンドゲーム用トーナメント
| ランク | 条件 | 特徴 |
|--------|------|------|
| **スターフォール（再戦）** | メインクリア後 | 対戦相手が大幅強化 |
| **裏ランク・真** | 裏ストーリー進行 | 変異種使いの対戦相手 |
| **無差別級** | 全トーナメント制覇後 | レベル制限なし、最強NPC集結 |
| **制限戦** | 特殊解放 | 種族縛り、属性縛り等のルール戦 |

## 6.6 図鑑コンプ・収集要素

### 図鑑コンプリート
- 400体全モンスターの発見・入手
- 多段配合が必要なモンスターは攻略情報なしでは困難
- 図鑑完成度に応じた報酬（段階的）
- 図鑑の価値は「400枠の多さ」だけでなく、変異録、配合履歴、地域差分の発見で体感的に拡張する

### 変異種コレクション
- 変異種は図鑑とは別の「変異録」に記録
- 全変異種の発見はやり込みの頂点
- 特定の変異種は特定条件でしか出現しない

### 称号システム
- 特定の実績で称号を獲得
- 称号に応じた恩恵（微量のステータスボーナス等、要検討）

## 6.7 周回・リプレイ要素

### 強くてニューゲーム（要検討）
- メインクリア後にLv1からやり直し、一部引き継ぎ
- 引き継ぎ候補: 図鑑データ、一部アイテム、称号
- 2周目以降で変化するイベント・台詞
- 隠しルート解放

### 日替わり/週替わりチャレンジ（要検討）
- オフラインでも日付ベースで変化するチャレンジ
- 特定条件下でのバトル・ダンジョン攻略
- 報酬: 限定アイテム、変異確率上昇等
