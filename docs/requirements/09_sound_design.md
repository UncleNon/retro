# 09. サウンド設計

> **ステータス**: Draft v1.0
> **最終更新**: 2026-03-15

## 9.1 設計方針

> GBCの8bit音源風サウンドをAI生成メインで制作。ノスタルジーを喚起しつつ、楽曲としての品質は現代水準を目指す。

|----------|---------|------|

| **フィールド** | 8〜12曲 | 世界テーマ別 |

| **合計** | **40〜60曲** | |

### BGMの音楽的方向性
| シーン | 音楽的特徴 |
|--------|-----------|
| 拠点 | 穏やか、安心感、各世界の文化を反映 |
| フィールド | 冒険感、世界テーマに合わせた楽器選択 |
| ダンジョン | 緊張感、不安、探索の高揚 |
| 通常バトル | テンポ良い、繰り返し聴いても飽きない |
| ボスバトル | 熱い、荘厳、プレッシャー |
| イベント（ダーク） | 不穏、不協和音、禁忌を匂わせる |

### BGM技術仕様
| 項目 | 仕様 |
|------|------|
| **音源スタイル** | 8bit風チップチューン（GBC PSG音源風） |
| **チャンネル数** | 4ch目安（矩形波×2 + 波形メモリ×1 + ノイズ×1） |
| **フォーマット** | WAV or OGG（Godot対応） |
| **ループ** | シームレスループ対応 |
| **クロスフェード** | 場面転換時0.5秒 |

## 9.5 AI生成方針

### 使用ツール候補
| ツール | 用途 | 備考 |
|--------|------|------|
| **Suno** | BGM生成 | チップチューン品質を評価して選定 |
| **Udio** | BGM生成 | Sunoと比較評価 |
| **その他新興ツール** | 随時評価 | AI音楽生成も急速進化中 |

### 生成プロンプト方針
- ジャンル指定: "8-bit chiptune", "Game Boy Color style music"
- ムード指定: シーンに合わせた感情キーワード
- テンポ指定: BPM目安を指定
- ループ対応の指示

### 品質管理
- 生成後にループポイントの手動調整
- 音量バランスの統一（ラウドネス正規化）
- ゲーム内での実際の聴感テスト
- 既存楽曲との類似チェック

## 9.6 音量管理

### 音量カテゴリ
| カテゴリ | デフォルト | ユーザー調整 |
|----------|-----------|-------------|
| マスター | 100% | 0〜100% |
| BGM | 70% | 0〜100% |
| SE | 100% | 0〜100% |

### 音量バランス基準
- BGMはSEの邪魔をしないレベルに
- 重要なSE（会心、レベルアップ等）はBGMの上に乗る
- クロスフェード時に音量ディップしない設計
