## 14.1 対応プラットフォーム

| 項目 | 仕様 |
|------|------|
| **プラットフォーム** | iOS |
| **最低対応バージョン** | iOS 16+（2026年時点で3世代前まで） |
| **対応デバイス** | iPhone SE (3rd gen)以降、iPad対応は要検討 |
| **画面方向** | 縦持ち固定（GBCの画面比率に最適） |

---

## 14.2 パフォーマンス要件

| 項目 | 要件 |
|------|------|
| **フレームレート** | 60fps安定（30fps下限） |
| **シーン遷移** | < 2秒 |
| **バトル開始** | < 1秒（エンカウントから戦闘画面表示まで） |
| **セーブ/ロード** | < 1秒 |
| **起動時間** | < 5秒（コールドスタート） |
| **メモリ使用量** | < 300MB（Godotは軽量） |
| **アプリサイズ** | < 300MB（Godotランタイムは軽量） |
| **バッテリー** | 連続プレイ3時間+（iPhone 15基準） |

## 14.3 オフライン対応

| 項目 | 仕様 |
|------|------|
| **基本方針** | 完全オフライン対応 |
| **オフラインで可能** | 全ゲーム機能（ストーリー、バトル、配合、トーナメント、全コンテンツ） |
| **オンライン必要** | なし（iCloud同期を採用する場合のみ任意） |
| **ネットワーク断時** | ゲームプレイに影響なし。クラウド同期採用時はネットワーク復帰後に再試行 |

## 14.4 セキュリティ要件

> **注意**: 以下は最終リリースの目標仕様。Phase 0 ではローカルセーブ単体で成立する構成を優先し、暗号化・Keychain連携は技術スパイク通過後に段階導入する（ADR-0010参照）。

### セーブデータ保護
| 項目 | 仕様 |
|------|------|
| **暗号化** | AES-256 |
| **キー管理** | iOS Keychain（ハードコード禁止） |
| **整合性** | SHA-256チェックサム |
| **改ざん検知** | チェックサム不一致時は破損扱い→バックアップから復元試行 |

### チート対策
| レベル | 対策 |
|--------|------|
| **基本** | セーブデータ暗号化 + 署名 |
| **中級** | メモリ上の重要値の難読化（XOR等） |
| **高級** | ステータス異常値検出（上限/下限チェック） |
| **運用** | 専用サーバーは持たない。整合性検証はクライアント側とCloudKitメタデータで行う |

### プライバシー
- 個人情報を一切収集しない（趣味プロジェクト）
- 分析SDK不使用
- 広告SDK不使用
- App Tracking Transparency不要

## 14.5 アクセシビリティ

### 対応項目
| 項目 | 対応 |
|------|------|
| **フォントサイズ** | GBC固定サイズ（レトロ感維持のため変更不可。ただしiPhone上での表示サイズは整数倍スケーリングで確保） |
| **色覚対応** | ステータスバーに数値表示を併記（色だけに依存しない） |
| **VoiceOver** | 基本対応（メニュー項目の読み上げ） |
| **ボタンサイズ** | バーチャルパッドのサイズ3段階調整 |
| **テキスト速度** | 表示速度調整可能 |
| **画面フラッシュ** | エフェクトの点滅を抑制するオプション |

---

## 14.6 安定性・信頼性

| 項目 | 要件 |
|------|------|
| **クラッシュ率** | < 0.1%（セッションあたり） |
| **データロスト** | ゼロを目標（オートセーブ+ローカルバックアップ。iCloud採用時はクラウドも併用） |
| **エラーハンドリング** | クラッシュ回避。エラー時はユーザーに通知+自動リカバリ試行 |
| **異常終了復帰** | オートセーブからの復帰 |

---

## 14.7 保守性

| 項目 | 方針 |
|------|------|
| **コンテンツ追加** | データ駆動設計により、コード変更なしでモンスター・世界を追加可能 |
| **バランス調整** | CSVマスターデータの編集→インポートで反映 |
| **バグ修正** | CIによるリグレッション防止 |
| **将来拡張** | オンライン対戦・交換を見据えたインターフェース設計 |

## 14.8 法的要件

| 項目 | 対応 |
|------|------|
| **App Store審査ガイドライン** | 準拠 |
| **プライバシーポリシー** | 必要（データ収集なしでも） |
| **年齢レーティング** | ダークファンタジー要素を考慮し、12+想定 |
| **著作権** | 全アセットがオリジナル or ライセンス取得済み |
| **商標** | ゲームタイトル・モンスター名の商標調査実施 |
| **EULA** | App Store標準 |
