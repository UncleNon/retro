# 3. 製品コンセプトとゲーム全体像

> **Archived**: この文書は旧設計。現行の source of truth は `docs/requirements/`。

本企画の仮題は『Project 継承獣クロニクル』とする。主人公は古びた牧場を継いだ見習い"継承士"。世界各地の『門印』が壊れたことで、各地域の生態と記録が断片化している。主人公は契約した継承獣を育て、6つの門印を修復しながら、記録改ざんを進める組織『黒鐘団』の目的を追う。基本体験は『短く探索する → 契約獣を増やす → 継承合成で系譜をつなぐ → 新地域へ進む』である。

| **ジャンル**       | モンスター育成 + 収集 + 合成を中核にしたトップダウンRPG                                 |
|--------------------|-----------------------------------------------------------------------------------------|
| **プレイ人数**     | 1人。オンライン対戦や交換は初版では非採用。                                             |
| **想定プレイ時間** | メイン30〜35時間、図鑑/希少合成/裏迷宮込みで45〜55時間。                                |
| **対象ユーザー**   | 初代携帯機RPGの感触を求める層、10〜20分単位で遊びたい層、育成計画を立てるのが好きな層。 |
| **世界構造**       | ハブ牧場 + 6地域 + 最終塔 + 裏迷宮。小さな画面単位で区切られたタイル型世界。            |
| **戦闘構造**       | 3体パーティ制ターンバトル。個別入力と戦術AIの切替が可能。                               |
| **固有のフック**   | 継承合成で『見た目は新種、系譜は手持ちの履歴を引き継ぐ』成長履歴を残す。                |

## 3.1 企画の三本柱

| **柱** | **定義**             | **要求される実装**                                                                   |
|--------|----------------------|--------------------------------------------------------------------------------------|
| P1     | 狭い画面でも迷わない | 地形・出口・イベント・NPCの優先度を一画面で整理。探索中の常時HUDは極小。             |
| P2     | 育成履歴が資産になる | 単純な捕獲ではなく、合成と継承で過去の手持ちが次世代の強みになる。                   |
| P3     | 10分でも進む         | 遭遇率・会話量・マップ密度・メニュー手数を調整し、短時間でも前進感がある設計にする。 |

> **非採用 / 初版で切るもの**
>
> オンライン対戦、交換、PvPランク、音声付き会話、オープンワールド、物理演算、手描き高解像度演出、自由視点3D、生成マップ、フルボイス、ガチャ。これらは"初代級の密度"を薄めるか、少人数開発で破綻しやすい。
